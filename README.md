# SBS게임아카데미 교육 자료
김진우 강사 강의 교육 자료

## 처음 시작하는 사람 추천 순서
1. 프로그래밍이 처음이면 `C -> C++ -> C#` 순서로 기초 문법을 학습
2. 게임 엔진 입문은 `Unity` 또는 `Unreal Engine` 중 1개를 먼저 선택
3. 엔진 1개 완주 후 나머지 엔진을 비교 학습

## 트랙 선택 가이드
- 프로그래밍 기초부터 탄탄히: `C`, `C++`, `C#`
- 결과물 제작 중심: `Unity`, `Unreal Engine`
