# SBS게임아카데미 교육 자료
김진우 강사 강의 교육 자료

## 처음 시작하는 사람 추천 순서
1. 프로그래밍이 처음이면 `C -> C++ -> C#` 순서로 기초 문법을 학습
2. 버전 관리 기초는 `Git` 또는 `SVN` 트랙을 선택해 병행 학습
3. 게임 엔진 입문은 `Unity` 또는 `Unreal Engine` 중 1개를 먼저 선택
4. 엔진 1개 완주 후 나머지 엔진을 비교 학습

## 트랙 선택 가이드
- 프로그래밍 기초부터 탄탄히: `C`, `C++`, `C#`
- 협업/이력 관리 역량 강화: `Git`, `SVN`
- 결과물 제작 중심: `Unity`, `Unreal Engine`
