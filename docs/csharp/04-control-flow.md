# Chapter 04 조건문과 반복문

## 학습 목표
- `if`, `switch`, `for`, `while`을 목적에 맞게 선택한다.
- `break`, `continue` 흐름을 이해한다.

## 세부 주제
### 04-1 조건문
- `if/else`, `switch`

### 04-2 반복문
- `for`, `while`

### 04-3 흐름 제어
- `break`, `continue`

## 실습 체크리스트
- 메뉴 선택형 콘솔 프로그램을 만든다.

## 본문

### 04-1 조건문
- `if/else`는 범위 조건에 적합합니다.
- `switch`는 고정된 선택지 분기에 적합합니다.

### 04-2 반복문
- 반복 횟수가 명확하면 `for`, 명확하지 않으면 `while`이 편합니다.
- 무한 루프를 만들지 않도록 종료 조건을 먼저 설계합니다.

### 04-3 흐름 제어
- `break`는 반복문을 즉시 종료합니다.
- `continue`는 현재 회차를 건너뛰고 다음 회차로 이동합니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 분기(branch) | 조건에 따라 실행 경로가 나뉘는 구조 |
| 반복(loop) | 같은 코드 블록을 조건이 참인 동안 반복 실행하는 구조 |

예시 코드:
```csharp
using System;

class Program
{
    static void Main()
    {
        for (int i = 1; i <= 10; i++)
        {
            if (i % 2 == 0) continue;
            Console.Write(i + " ");
        }
    }
}
```

예상 출력:
```text
1 3 5 7 9
```

코드 해석:
- `for`문으로 1~10을 순회하고 짝수는 `continue`로 건너뜁니다.
- 조건 통과한 홀수만 출력해 흐름 제어 키워드 동작을 확인할 수 있습니다.

연습문제:
1. 문제 1 - 구구단
   - 문제: 단을 입력받아 해당 단을 출력하세요.
   - 입력: 2~9
   - 출력: 곱셈식 9줄
   - 조건: `for` 사용
2. 문제 2 - 메뉴 분기
   - 문제: 메뉴 번호에 따라 다른 문장을 출력하세요.
   - 입력: 메뉴 번호 1개
   - 출력: 결과 메시지
   - 힌트: `switch`의 `default` 포함

정답 포인트:
- 반복/분기 문법보다 종료 조건 설계가 더 중요
- `break`와 `continue` 역할을 구분
