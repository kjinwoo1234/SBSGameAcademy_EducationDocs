# Chapter 21 콘솔 제어로 뱀/테트리스 만들기

## 학습 목표
- C# 콘솔에서 커서 이동/텍스트 색상/화면 지우기를 제어할 수 있다.
- 뱀게임과 테트리스의 공통 게임 루프를 구현할 수 있다.

## 세부 주제
### 21-1 콘솔 제어 함수
- `SetCursorPosition`, `ForegroundColor`, `Clear`

### 21-2 뱀게임 루프 설계
- 입력, 이동, 충돌, 렌더링

### 21-3 테트리스 루프 설계
- 낙하, 충돌, 고정, 줄 삭제

## 실습 체크리스트
- 콘솔 제어 유틸 클래스를 만든 뒤 두 게임 공통으로 사용한다.

## 본문

### 21-1 콘솔 제어 함수
- C# 콘솔은 기본 API만으로 좌표 기반 UI를 만들 수 있습니다.
- `Console.SetCursorPosition`으로 위치를 이동하고 `Console.ForegroundColor`로 색을 바꿉니다.
- `Console.Clear`는 전체 화면을 지우는 가장 단순한 방법입니다.

함수 비교:
| 기능 | C# 콘솔 함수 | 용도 |
|---|---|---|
| 커서 이동 | `Console.SetCursorPosition(x, y)` | 좌표 출력 |
| 텍스트 색상 | `Console.ForegroundColor` | 상태 강조 |
| 화면 지우기 | `Console.Clear()` | 화면 초기화 |

용어 설명:
| 용어 | 의미 |
|---|---|
| 게임 루프(game loop) | 입력 처리 -> 상태 갱신 -> 출력 반복 구조 |
| 상태 갱신(update) | 현재 프레임에서 게임 데이터(좌표/점수)를 계산하는 단계 |

예시 코드:
```csharp
using System;

class ConsoleUtil
{
    public static void MoveTo(int x, int y) => Console.SetCursorPosition(x, y);
    public static void SetColor(ConsoleColor color) => Console.ForegroundColor = color;
    public static void Clear() => Console.Clear();
}
```

예상 결과:
```text
좌표 출력, 텍스트 색상 변경, 화면 지우기 동작 확인
```

코드 해석:
- 유틸 함수로 감싸두면 게임 로직에서 콘솔 API 의존을 줄일 수 있습니다.
- 색상 설정은 UI 피드백(점수/경고/게임오버) 표현에 매우 유용합니다.

### 21-2 뱀게임 루프 설계
- 방향 입력을 읽고 머리 좌표를 갱신한 뒤, 벽/몸통 충돌 여부를 검사합니다.
- 먹이 충돌 시 길이 증가와 점수 갱신을 처리합니다.

### 21-3 테트리스 루프 설계
- 일정 시간 간격으로 블록을 하강시킵니다.
- 충돌 시 보드에 고정하고 완성된 줄을 삭제한 뒤 점수를 반영합니다.

연습문제:

2. 문제 2 - 뱀게임 기본 이동

   - 입력: 방향키
   - 출력: 이동 좌표
   - 조건: 틱 루프(`Thread.Sleep`) 사용
3. 문제 3 - 테트리스 줄 삭제
   - 문제: 꽉 찬 줄을 감지해 삭제하는 함수를 작성하세요.
   - 입력: 보드 2차원 배열
   - 출력: 줄 삭제 후 보드
   - 힌트: 아래 줄부터 검사

정답 포인트:
- 콘솔 제어 함수는 게임별 복붙이 아니라 공통 유틸로 분리
- 뱀/테트리스 모두 루프 구조는 동일하고 업데이트 규칙만 다름
- 깜빡임 최소화를 위해 전체 Clear 대신 부분 갱신도 고려 가능
