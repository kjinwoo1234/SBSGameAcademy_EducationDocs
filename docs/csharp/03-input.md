# Chapter 03 입력

## 학습 목표
- `Console.ReadLine()`으로 키보드에서 한 줄을 받을 수 있다.
- 받은 내용은 **문자열**이며, 숫자 계산 전에 변환이 필요함을 이해한다.
- 앞 장에서 배운 변수에 입력을 담아 출력·계산에 쓸 수 있다.

## 본문

### 03-1 키보드로 한 줄 받기

키보드로 한 줄을 받을 때는 **`Console.ReadLine()`**을 씁니다. 괄호 안에 넣을 글자가 없어도 `()`는 그대로 둡니다. 사용자가 Enter를 누르면, 그때까지 친 내용이 **문자열**로 돌아옵니다.

받은 내용은 변수에 담아 두면 다시 출력하거나 다음에 쓸 수 있습니다. 안내 문구는 앞 장의 `Console.Write("...")`로 같은 줄에 보여 주는 패턴이 흔합니다.

아래를 `Main`에 넣고, 콘솔에 이름을 입력한 뒤 출력을 확인해 보세요.

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("이름 입력: ");
        string name = Console.ReadLine();
        Console.WriteLine($"안녕하세요, {name}님!");
    }
}
```

**예상 출력 (이름을 `Jinwoo`로 입력한 경우)**

```text
이름 입력: Jinwoo
안녕하세요, Jinwoo님!
```

**코드 해석**
- `ReadLine()` 결과가 `name`에 문자열로 들어갑니다.
- `$"..."`로 인사 문장에 이름을 넣습니다.

### 03-2 숫자로 바꿔 쓰기

화면에 “나이를 입력하세요”라고 해도, 키보드로 친 `20`은 일단 **글자 `"20"`**입니다. 더하기·비교 같은 **숫자 계산**을 하려면 글자를 숫자로 바꿔야 합니다.

입문에서는 **`int.Parse("글자")`**로 문자열을 정수로 바꿉니다. `"20"`처럼 **숫자로 읽을 수 있는 글자**를 넣었을 때 사용합니다. `"abc"`처럼 숫자가 아닌 글자를 넣으면 프로그램이 오류로 멈출 수 있으므로, **연습할 때는 숫자만** 입력하세요. (잘못된 입력을 부드럽게 다루는 방법은 조건문을 배운 뒤 이어서 다룹니다.)

**오류 방지 팁 (숫자 변환 실패)**
- **언제:** `"abc"`처럼 숫자가 아닌 글자를 `Parse`로 바꾸려 할 때
- **어떤 일:** 프로그램이 갑자기 멈출 수 있음
- **어떻게:** 지금은 숫자만 입력한다. 실패를 나눠 처리하는 방법은 이후 장에서 이어서 배운다

아래를 `Main`에 넣고, 나이를 **숫자**로 입력한 뒤 출력을 확인해 보세요.

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("나이 입력: ");
        string ageText = Console.ReadLine();
        int age = int.Parse(ageText);

        Console.WriteLine($"당신은 {age}살입니다.");
        Console.WriteLine($"10년 뒤: {age + 10}");
    }
}
```

**예상 출력 (정상 입력 예)**

```text
나이 입력: 20
당신은 20살입니다.
10년 뒤: 30
```

**코드 해석**
- 나이를 먼저 **문자열**로 받고, `int.Parse`로 **정수** `age`로 바꿉니다.
- 그다음 `age + 10`처럼 숫자 계산이 가능합니다.
- 변환 없이 `ageText + 10`을 하면 글자 연결이 될 수 있습니다.

### 연습문제

**문제 1**
- 문제: 이름을 입력받아 `반가워요, (이름)!`을 출력하세요.
- 입력: 이름 한 줄 (예: `Kim`)
- 출력: 예) `반가워요, Kim!`
- 조건: `ReadLine` + 문자열 보간

**문제 2**
- 문제: 정수 하나를 입력받아 2배 값을 출력하세요.
- 입력: 정수(문자열로 입력, 예: `5`)
- 출력: 예) `10`
- 조건: `int.Parse` 사용

**문제 3**
- 문제: 정수 2개를 입력받아 합계를 출력하세요.
- 입력: 정수 2개(각각 Enter, 예: `3` 다음 `7`)
- 출력: 예) `10`
- 조건: 각각 `int.Parse`로 변환한 뒤 더할 것. 변환 전 `+`는 글자 연결이 될 수 있음

### 정답 포인트

- 문제 1: `string name = Console.ReadLine();` → `WriteLine($"반가워요, {name}!");`
- 문제 2: `int n = int.Parse(Console.ReadLine());` → `n * 2` 출력
- 문제 3: 두 번 읽고 각각 `int.Parse` 후 `a + b`
- 공통: 키보드 입력의 기본은 문자열, 숫자 계산 전에 변환
