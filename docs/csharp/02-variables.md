# Chapter 02 변수와 자료형

## 학습 목표
- C# 기본 자료형과 변수 선언 방식을 익힌다.
- `var`와 명시적 타입 선언의 차이를 이해한다.

## 세부 주제
### 02-1 기본 자료형
- `int`, `double`, `bool`, `char`, `string`

### 02-2 변수 선언/초기화
- 선언, 대입, 재할당

### 02-3 `var`
- 타입 추론

## 실습 체크리스트
- 같은 값을 서로 다른 자료형에 저장해 출력 비교를 해본다.

## 본문

### 02-1 기본 자료형
- 자료형은 변수에 어떤 종류의 값을 저장할지 결정합니다.
- 숫자, 문자, 논리값, 문자열은 가장 먼저 익혀야 할 핵심 타입입니다.

### 02-2 변수 선언/초기화
- 변수는 `타입 이름 = 값;` 형태로 선언합니다.
- 선언 후 값을 다시 넣는 재할당도 가능합니다.

### 02-3 `var`
- `var`는 우변 값을 보고 타입을 추론합니다.
- 타입이 분명할 때는 가독성을 높일 수 있지만 남용은 피하는 것이 좋습니다.

자료형 표:
| 타입 | 예시 값 | 설명 |
|---|---|---|
| `int` | `10` | 정수 |
| `double` | `3.14` | 실수 |
| `bool` | `true` | 참/거짓 |
| `char` | `'A'` | 문자 1개 |
| `string` | `"Hello"` | 문자열 |

용어 설명:
| 용어 | 의미 |
|---|---|
| 자료형(type) | 변수에 저장할 값의 종류와 범위를 정하는 규칙 |
| 타입 추론(type inference) | 초기값을 기준으로 타입을 자동으로 결정하는 기능 |

예시 코드:
```csharp
using System;

class Program
{
    static void Main()
    {
        int level = 5;
        double ratio = 0.8;
        var name = "Kim"; // string으로 추론
        Console.WriteLine($"{name}, level={level}, ratio={ratio}");
    }
}
```

예상 출력:
```text
Kim, level=5, ratio=0.8
```

코드 해석:
- `var name = "Kim";`은 초기값을 기준으로 `string` 타입으로 결정됩니다.
- 서로 다른 타입 변수를 한 문장에 출력해 자료형 차이를 확인합니다.

연습문제:
1. 문제 1 - 기본 타입 출력
   - 문제: `int`, `double`, `bool` 변수를 선언하고 출력하세요.
   - 입력: 없음
   - 출력: 변수 값
   - 조건: 각 타입 1개 이상
2. 문제 2 - var 사용
   - 문제: `var`로 문자열과 정수를 각각 선언해 출력하세요.
   - 입력: 없음
   - 출력: 선언한 값 2개
   - 힌트: 초기값으로 타입 결정

정답 포인트:
- C#은 타입 안정성을 갖는 언어
- `var`도 컴파일 시점에 타입이 고정됨
