# C# 자습 자료

## 권장 대상
- 객체지향 문법을 기반으로 게임/응용 개발을 시작하려는 학습자
- Unity 스크립팅 전 C# 핵심 문법을 체계적으로 배우려는 학습자

## 선수지식
- 변수/조건문/반복문/함수 기초
- 클래스 개념을 간단히 들어본 경험(없어도 학습 가능)

## 완주 후 역량
- C#의 객체지향 문법(클래스, 상속, 다형성)을 활용할 수 있다.
- 델리게이트/이벤트/비동기를 포함한 실전형 기초 코드를 작성할 수 있다.
