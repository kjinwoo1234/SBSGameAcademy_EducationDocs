# Chapter 03 연산자

## 학습 목표
- 산술/비교/논리 연산자를 구분해 사용할 수 있다.
- 형 변환이 필요한 상황을 이해한다.

## 세부 주제
### 03-1 산술/비교 연산자
- 계산과 조건식

### 03-2 논리 연산자
- `&&`, `||`, `!`

### 03-3 형 변환
- `(double)`, `Convert`, `Parse`

## 실습 체크리스트
- 정수 나눗셈과 실수 나눗셈 결과 차이를 확인한다.

## 본문

### 03-1 산술/비교 연산자
- 산술 연산자는 수치 계산에 사용됩니다.
- 비교 연산자는 조건식에서 참/거짓 결과를 만듭니다.

### 03-2 논리 연산자
- `&&`는 모두 참일 때 참, `||`는 하나라도 참이면 참입니다.
- `!`는 논리값을 반전합니다.

### 03-3 형 변환
- 정수끼리 나누면 소수점이 버려집니다.
- 실수 계산이 필요하면 명시적으로 `double`로 변환합니다.

연산자 요약:
| 분류 | 연산자 | 용도 |
|---|---|---|
| 산술 | `+ - * / %` | 계산 |
| 비교 | `== != > < >= <=` | 조건식 |
| 논리 | `&& || !` | 조건 결합 |

용어 설명:
| 용어 | 의미 |
|---|---|
| 명시적 형 변환 | 개발자가 타입 변환을 코드로 직접 지정하는 방식 |
| 암시적 형 변환 | 컴파일러가 자동으로 수행하는 타입 변환 |

예시 코드:
```csharp
using System;

class Program
{
    static void Main()
    {
        int a = 7, b = 2;
        Console.WriteLine(a / b);
        Console.WriteLine((double)a / b);
    }
}
```

예상 출력:
```text
3
3.5
```

코드 해석:
- `a / b`는 정수 나눗셈이라 소수점이 버려집니다.
- `(double)a / b`는 실수 계산으로 바뀌어 소수 결과가 유지됩니다.

연습문제:
1. 문제 1 - 평균 계산
   - 문제: 정수 3개의 평균을 소수점으로 출력하세요.
   - 입력: 정수 3개
   - 출력: 평균
   - 조건: 형 변환 사용
2. 문제 2 - 합격 판정
   - 문제: 점수와 출석률을 입력받아 합격 여부를 출력하세요.
   - 입력: 점수, 출석률
   - 출력: 합격/불합격
   - 힌트: 논리 연산자 사용

정답 포인트:
- 정수 나눗셈은 소수점 손실 발생
- 조건 결합은 괄호로 의도 명확화
