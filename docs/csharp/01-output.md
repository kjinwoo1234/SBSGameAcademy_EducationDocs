# Chapter 01 출력과 입력

## 학습 목표
- C# 콘솔 프로그램의 기본 입출력 흐름을 이해한다.
- `Console.WriteLine`과 `Console.ReadLine`의 역할을 구분할 수 있다.

## 세부 주제
### 01-1 C# 프로그램 시작
- `Main` 메서드

### 01-2 출력
- `Console.WriteLine`

### 01-3 입력
- `Console.ReadLine`

## 실습 체크리스트
- 이름을 입력받아 인사 문장을 출력해본다.

## 본문

### 01-1 C# 프로그램 시작
- C# 콘솔 앱은 `Main` 메서드에서 실행을 시작합니다.
- `using System;`을 선언하면 콘솔 입출력 기능을 사용할 수 있습니다.

### 01-2 출력
- `Console.WriteLine`은 줄바꿈과 함께 텍스트를 출력합니다.
- 문자열 보간(`$"..."`)을 사용하면 변수 값을 문장에 쉽게 넣을 수 있습니다.

### 01-3 입력
- `Console.ReadLine`은 한 줄 입력을 문자열로 반환합니다.
- 숫자 입력은 문자열을 정수로 변환하는 과정이 필요합니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 보간 문자열 | 문자열 안에 변수 값을 직접 삽입하는 문법(`$"{name}"`) |
| 파싱(parsing) | 문자열을 숫자 같은 다른 타입으로 해석해 바꾸는 과정 |

예시 코드:
```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("이름 입력: ");
        string name = Console.ReadLine() ?? "";
        Console.WriteLine($"안녕하세요, {name}님!");
    }
}
```

예상 출력:
```text
이름 입력: Jinwoo
안녕하세요, Jinwoo님!
```

코드 해석:
- `ReadLine()` 결과가 `null`일 수 있어 `?? ""`로 빈 문자열을 대체합니다.
- `$"..."` 문법으로 문자열과 변수를 자연스럽게 결합해 출력합니다.

연습문제:
1. 문제 1 - 나이 출력
   - 문제: 나이를 입력받아 "당신은 n살입니다."를 출력하세요.
   - 입력: 나이 1개
   - 출력: 안내 문장 1줄
   - 조건: `int.Parse` 또는 `int.TryParse` 사용
2. 문제 2 - 두 수 합
   - 문제: 정수 2개를 입력받아 합계를 출력하세요.
   - 입력: 정수 2개
   - 출력: 합계 1개
   - 힌트: 입력은 문자열, 계산은 정수

정답 포인트:
- 콘솔 입력은 기본적으로 문자열
- 숫자 계산 전 변환 단계가 필요
