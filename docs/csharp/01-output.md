# Chapter 01 출력

## 학습 목표

- 프로그램이 무엇인지, 코드를 “실행한다”는 말이 무엇인지 대략 설명할 수 있다.
- `Console.WriteLine("문장")` / `Console.Write("문장")`으로 화면에 글자를 출력할 수 있다.
- `Write`와 `WriteLine`의 줄바꿈 차이를 구분해 쓸 수 있다.

## 본문

### 01-1 프로그램이 시작되는 곳

**프로그램**은 컴퓨터에게 “이렇게 해 줘”라고 적어 둔 지시 모음입니다. 그 지시를 사람이 읽고 쓸 수 있게 적어 둔 글이 **코드**이고, Visual Studio(또는 비슷한 도구)에서 **실행** 버튼을 누르면 컴퓨터가 그 코드를 **위에서 아래 순서**로 따라 합니다. Windows 같은 **운영체제**는 컴퓨터의 기본 환경이고, 우리가 만든 프로그램을 “켜 주는” 역할을 합니다. “실행한다”는 말은 곧 **적어 둔 코드 한 줄 한 줄을 실제로 수행한다**는 뜻에 가깝습니다.

C# 콘솔에서는 보통 `Main`이라고 적힌 부분 **안쪽**부터 따라가기 시작합니다. **우리가 쓰는 문장은** `Main`**의** `{`**와** `}` **사이에 적는다**고만 기억하면 충분합니다.

**따라 실행하기 (처음 한 번)**

1. Visual Studio에서 **콘솔 앱** 프로젝트를 만든다. (이미 있으면 그 프로젝트를 연다.)
2. `Program` 파일의 `Main` `{` `}` **안**을 아래 코드처럼 맞춘다.
3. **디버그하지 않고 실행**(보통 Ctrl+F5)한다. 창이 바로 닫히면 출력을 못 볼 수 있다.
4. 검은 콘솔 창에 글이 보이면 성공이다. `예상 출력`과 다르면 따옴표·`;`를 다시 본다.

지금부터 가장 간단한 코드 하나를 만들어 보겠습니다. Visual Studio에서 실행하면 콘솔 창에 한 줄이 출력됩니다. 아래에는 `class`, `static`, `void` 같은 낯선 단어도 보이는데, **지금은 뜻을 외울 필요 없습니다.** “콘솔 프로그램의 **기본 뼈대**가 이렇게 생겼다”고만 보고, 우리가 고칠 곳은 `Main`의 `{` `}` **안**이라고 생각하면 됩니다.

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("프로그램이 시작되었습니다.");
    }
}
```

**예상 출력**

```text
프로그램이 시작되었습니다.
```

맨 위 `using System;`은 C#이 준비한 기능을 쓰기 위해 자주 들어가는 줄입니다. Visual Studio가 새 프로젝트를 만들 때 자동으로 넣어 주는 경우가 많고, 관련 개념은 나중에 따로 다룹니다. 지금은 **지우지 말고 그대로 두면** 됩니다. 같은 이유로 `class Program`, `static void Main()`도 “나중에 배울 단어가 붙은 **기본 형태**”로만 두고, 우선은 안쪽에 출력 문장을 적어 실행해 보는 데 집중하세요.

**코드 해석**

- `Main`의 `{` `}` 안에 있는 줄이 실제로 실행됩니다.
- `Console.WriteLine("...");` 한 줄이 화면(검은 콘솔 창)에 문장을 보여 줍니다.



### 01-2 화면에 글자 출력하기

화면에 글자를 **출력**할 때는 `Console.WriteLine("출력할 글자")` 형태를 가장 먼저 익히면 됩니다. 괄호 안의 **큰따옴표** `"..."`**로 묶인 글**을 **문자열(문장·글자 덩어리)**이라고 부르고, 그 내용이 콘솔에 나타납니다. 줄 끝의 `;`는 “이 지시가 여기서 끝난다”는 표시입니다.

**`Console.WriteLine("문장")`**은 문장을 출력한 뒤 **줄을 바꿉니다.**  
**`Console.Write("문장")`**은 출력한 뒤 **같은 줄에 이어서** 쓸 수 있습니다. 한 줄에 여러 조각을 이어 붙일 때는 `Write`, 한 줄을 마칠 때는 `WriteLine`을 자주 씁니다.


| 쓰는 형태                     | 줄바꿈 | 이럴 때         |
| ------------------------- | --- | ------------ |
| `Console.Write("문장")`     | 없음  | 같은 줄에 이어서 출력 |
| `Console.WriteLine("문장")` | 있음  | 결과 한 줄 출력    |


`Console.WriteLine();`처럼 괄호 안을 비우면 **줄바꿈만** 합니다. 글자는 출력하지 않습니다.

이 장에서는 **따옴표 안에 적어 둔 문장을 그대로 출력**하는 데만 집중합니다. 값을 이름 붙여 담아 두는 법과, 키보드로 글을 받는 법은 **다음 장들**에서 이어집니다.

아래 코드를 실행해 `Write`와 `WriteLine` 차이를 확인해 보세요.

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("닉네임: ");
        Console.Write("Jinwoo");
        Console.WriteLine();
        Console.WriteLine("점수: 100");
    }
}
```

**예상 출력**

```text
닉네임: Jinwoo
점수: 100
```

**코드 해석**

- `Console.Write("닉네임: ");`는 따옴표 안 글자만 출력하고 줄은 바꾸지 않습니다.
- 이어서 `Write("Jinwoo");`로 같은 줄에 이름을 붙입니다.
- `Console.WriteLine();`만 쓰면 **줄바꿈만** 합니다.
- `Console.WriteLine("점수: 100");`은 문장을 출력한 뒤 줄을 바꿉니다.



### 연습문제

**문제 1**

- 문제: `Console.WriteLine`으로 `Hello, C#`을 출력하세요.
- 입력: 없음
- 출력: `Hello, C#`
- 조건: 큰따옴표로 문장을 묶을 것

**문제 2**

- 문제: `Write`로 `이름:` 을 출력한 뒤, 같은 줄에 `Kim`을 이어서 출력하세요. 다음 줄에는 `완료`를 출력하세요.
- 입력: 없음
- 출력: 첫 줄 `이름: Kim`, 다음 줄 `완료`
- 조건: `Console.Write`와 `Console.WriteLine`을 모두 사용

**문제 3**

- 문제: 세 줄로 `하나`, `둘`, `셋`을 각각 출력하세요.
- 입력: 없음
- 출력: 첫 줄 `하나`, 둘째 줄 `둘`, 셋째 줄 `셋`
- 조건: `Console.WriteLine`을 세 번 사용



### 정답 포인트

- 문제 1: `Console.WriteLine("Hello, C#");`
- 문제 2: `Console.Write("이름: ");` → `Console.Write("Kim");` → 줄바꿈 → `Console.WriteLine("완료");`
- 문제 3: `Console.WriteLine("하나");` `Console.WriteLine("둘");` `Console.WriteLine("셋");`
- 공통: 출력은 큰따옴표로 묶은 문장으로 한다
