# Chapter 07 클래스와 객체

## 학습 목표
- 클래스와 객체의 관계를 이해한다.
- 필드/메서드로 상태와 동작을 설계할 수 있다.

## 세부 주제
### 07-1 클래스 기본
- 필드와 메서드

### 07-2 생성자
- 객체 초기화

### 07-3 접근 제한자
- `public`, `private`

## 실습 체크리스트
- `Player` 클래스를 만들고 상태를 변경하는 메서드를 작성한다.

## 본문

### 07-1 클래스 기본
- 클래스는 데이터와 기능을 하나로 묶는 설계도입니다.
- 객체는 클래스를 기반으로 생성된 실제 인스턴스입니다.

### 07-2 생성자
- 생성자는 객체 생성 시 자동 호출됩니다.
- 초기 상태를 안전하게 설정하는 역할을 합니다.

### 07-3 접근 제한자
- `public`은 외부 접근 허용, `private`는 클래스 내부 전용입니다.
- 내부 필드는 `private`로 보호하고 메서드로 제어하는 패턴이 기본입니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 인스턴스(instance) | 클래스로부터 생성된 실제 객체 |
| 캡슐화(encapsulation) | 데이터와 동작을 묶고 내부 상태를 보호하는 설계 방식 |

예시 코드:
```csharp
using System;

class Player
{
    private int hp;
    public Player(int hp) { this.hp = hp; }
    public void Damage(int d) { hp -= d; }
    public int GetHp() { return hp; }
}

class Program
{
    static void Main()
    {
        Player p = new Player(100);
        p.Damage(15);
        Console.WriteLine(p.GetHp());
    }
}
```

예상 출력:
```text
85
```

코드 해석:
- 생성자로 초기 체력을 설정하고 `Damage` 메서드로 상태를 변경합니다.
- `hp` 필드는 `private`로 보호하고 `GetHp`로 읽기만 허용합니다.
