# Chapter 08 메서드

## 학습 목표
- 메서드 선언/호출/반환 구조를 이해한다.
- 매개변수 전달 방식(값 전달, `ref`, `out`)을 구분한다.
- 같은 이름·다른 시그니처의 **오버로딩**을 구분해 호출한다.

## 본문

### 08-1 메서드 기본

**메서드(method)**는 반복되는 코드를 **이름** 붙여 묶어 두고, 필요할 때 **호출**해 재사용하는 단위입니다. 형태는 `반환형 이름(매개변수)`입니다. 예: `static int Add(int a, int b)`는 정수 두 개를 받아 정수를 돌려줍니다. 반환형이 **`void`**이면 값을 반환하지 않고, 안에서 `Console.WriteLine`처럼 일만 합니다. 호출은 `Add(3, 4)`처럼 **이름(인자)** 입니다. 괄호 안 값을 **인자**, 선언부 칸을 **매개변수**라고 부릅니다.

### 08-2 매개변수 전달

기본은 **값 전달**입니다. 메서드에 넘긴 복사본이 바뀌어도 호출쪽 변수는 그대로입니다. 호출쪽 변수를 메서드가 직접 바꾸려면 **`ref`**(호출 전 초기화 필요) 또는 **`out`**(메서드 안에서 반드시 대입)을 씁니다. 선언과 호출 **양쪽**에 `ref`/`out`을 적어야 합니다.

| 방식 | 의미 | 이럴 때 | 주의점 |
|---|---|---|---|
| 값 전달 | 복사본 전달 | 원본을 바꾸지 않을 때 | 메서드 안 대입이 원본에 안 보임 |
| `ref` | 같은 변수 공유 | 기존 값을 읽어 바꾼 뒤 되돌릴 때 | 호출 전 변수 초기화 필요 |
| `out` | 결과 칸으로 사용 | 메서드가 값을 “채워” 돌려줄 때 | 메서드 안에서 대입 필수 |

### 08-3 메서드 오버로딩

**오버로딩(overloading)**은 같은 이름 메서드를 **시그니처**(이름 + 매개변수 타입·개수)가 다르면 여러 개 둘 수 있는 기능입니다. `Add(int, int)`와 `Add(double, double)`처럼 인자 타입에 맞는 버전이 선택됩니다. **반환형만** 다르게 해서 오버로딩할 수는 없습니다.

| 용어 | 의미 |
|---|---|
| 메서드 시그니처 | 메서드를 구분하는 정보(이름, 매개변수 타입/개수) |
| 오버로딩(overloading) | 같은 이름 메서드를 시그니처로 구분해 여러 개 정의하는 기능 |

아래 코드를 실행해 오버로딩 선택과 `ref`로 원본이 바뀌는 것을 확인해 보세요.

```csharp
using System;

class Program
{
    static int Add(int a, int b)
    {
        return a + b;
    }

    static double Add(double a, double b)
    {
        return a + b;
    }

    static void DoubleValue(ref int n)
    {
        n = n * 2;
    }

    static void Main()
    {
        Console.WriteLine(Add(3, 4));
        Console.WriteLine(Add(1.5, 2.5));

        int x = 5;
        DoubleValue(ref x);
        Console.WriteLine(x);
    }
}
```

**예상 출력**

```text
7
4
10
```

**코드 해석**
- `Add(3, 4)`는 `int` 버전, `Add(1.5, 2.5)`는 `double` 버전이 선택됩니다.
- `DoubleValue(ref x)`는 `x`와 같은 변수를 공유하므로 `5`가 `10`으로 바뀝니다.
- 값 전달만 썼다면 `Main`의 `x`는 `5`로 남습니다.

### 연습문제

**문제 1**
- 문제: `Print` 메서드를 문자열 버전과 정수 버전으로 오버로딩하세요.
- 입력: 없음(코드에서 `Print("hi");` `Print(7);` 호출)
- 출력:
  ```text
  hi
  7
  ```
- 조건: 메서드 이름 `Print` 동일, 매개변수 타입만 다름

**문제 2**
- 문제: `ref`로 전달한 정수를 2배로 바꾸는 메서드를 작성하세요.
- 입력: 없음(예: 변수 `n = 6` 후 메서드 호출)
- 출력: `12`
- 조건: 선언·호출 모두 `ref` 사용

### 정답 포인트

- 문제 1: `static void Print(string s)`, `static void Print(int n)` — 시그니처로 구분
- 문제 2: `void Scale(ref int n) { n *= 2; }` 후 `Scale(ref n);`
- 공통: 원본 변경이 필요하면 `ref`/`out`, 아니면 값 전달
