# Chapter 05 배열

## 학습 목표
- 1차원 배열 선언/초기화/순회를 익힌다.
- 인덱스 범위와 예외 위험을 이해한다.

## 세부 주제
### 05-1 배열 선언
- 크기 지정과 초기값

### 05-2 인덱스 접근
- `0`부터 시작

### 05-3 순회
- `for`, `foreach`

## 실습 체크리스트
- 점수 배열 평균과 최대값을 구한다.

## 본문

### 05-1 배열 선언
- 배열은 같은 타입 데이터를 연속된 공간에 저장합니다.
- `int[] scores = new int[5];`처럼 선언합니다.

### 05-2 인덱스 접근
- 인덱스는 `0`부터 시작합니다.
- 범위를 벗어나면 `IndexOutOfRangeException`이 발생합니다.

### 05-3 순회
- `for`는 인덱스 제어에 유리합니다.
- `foreach`는 읽기 중심 순회에 간결합니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 인덱스(index) | 배열에서 각 원소의 위치 번호 |
| 예외(exception) | 프로그램 실행 중 발생한 오류 상황 |

예시 코드:
```csharp
using System;

class Program
{
    static void Main()
    {
        int[] scores = { 80, 95, 70, 100 };
        int sum = 0;
        foreach (int s in scores) sum += s;
        Console.WriteLine((double)sum / scores.Length);
    }
}
```

예상 출력:
```text
86.25
```

코드 해석:
- `foreach`로 모든 점수를 순회해 합계를 계산합니다.
- `scores.Length`로 원소 개수를 구해 평균을 계산하고, 실수 나눗셈으로 출력합니다.

연습문제:
1. 문제 1 - 최대값 찾기
   - 문제: 정수 배열에서 최대값을 출력하세요.
   - 입력: 정수 n개
   - 출력: 최대값 1개
   - 조건: 반복문으로 직접 비교
2. 문제 2 - 짝수 개수
   - 문제: 배열에서 짝수 원소 개수를 구하세요.
   - 입력: 정수 배열
   - 출력: 짝수 개수
   - 힌트: `% 2 == 0` 조건

정답 포인트:
- 인덱스는 `0` 시작
- 평균 계산은 형 변환으로 정밀도 유지
