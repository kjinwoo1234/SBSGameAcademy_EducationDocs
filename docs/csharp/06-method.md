# Chapter 06 메서드

## 학습 목표
- 메서드 선언/호출/반환 구조를 이해한다.
- 매개변수 전달 방식(`value`, `ref`, `out`)을 구분한다.

## 세부 주제
### 06-1 메서드 기본
- 반환형, 이름, 매개변수

### 06-2 매개변수 전달
- 값 전달과 참조 전달

### 06-3 메서드 오버로딩
- 같은 이름, 다른 시그니처

## 실습 체크리스트
- 두 수 계산 기능을 메서드로 분리한다.

## 본문

### 06-1 메서드 기본
- 메서드는 반복 코드를 이름 붙여 재사용하는 단위입니다.
- 반환형이 `void`면 값을 반환하지 않습니다.

### 06-2 매개변수 전달
- 기본은 값 전달이라 원본이 바뀌지 않습니다.
- `ref`/`out`을 사용하면 호출자 변수에 결과를 반영할 수 있습니다.

### 06-3 메서드 오버로딩
- C#은 매개변수 타입/개수가 다르면 같은 이름 메서드를 여러 개 정의할 수 있습니다.
- 반환형만 다르게 해서 오버로딩할 수는 없습니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 메서드 시그니처 | 메서드를 구분하는 정보(이름, 매개변수 타입/개수) |
| 오버로딩(overloading) | 같은 이름 메서드를 시그니처로 구분해 여러 개 정의하는 기능 |

예시 코드:
```csharp
using System;

class Program
{
    static int Add(int a, int b) => a + b;
    static double Add(double a, double b) => a + b;

    static void Main()
    {
        Console.WriteLine(Add(3, 4));
        Console.WriteLine(Add(1.5, 2.5));
    }
}
```

예상 출력:
```text
7
4
```

코드 해석:
- 같은 이름 `Add`를 `int`와 `double` 버전으로 각각 정의했습니다.
- 호출 인자 타입에 맞는 메서드가 자동 선택되어 실행됩니다.

연습문제:
1. 문제 1 - 오버로딩 구현
   - 문제: `Print` 메서드를 문자열/정수 버전으로 오버로딩하세요.
   - 입력: 문자열 1개, 정수 1개
   - 출력: 각 값 출력
   - 조건: 메서드 이름 동일
2. 문제 2 - ref 사용
   - 문제: `ref`로 전달한 정수를 2배로 바꾸는 메서드를 작성하세요.
   - 입력: 정수 1개
   - 출력: 변경 후 값
   - 힌트: 호출부와 선언부 모두 `ref`

정답 포인트:
- 메서드 시그니처가 오버로딩 구분 기준
- 원본 변경이 필요하면 `ref/out` 고려
