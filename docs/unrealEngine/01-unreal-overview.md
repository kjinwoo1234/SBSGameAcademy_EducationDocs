## 학습 목표
- Unreal Engine의 기본 구조를 이해한다.
- 새 프로젝트를 생성하고 실행할 수 있다.

## 세부 주제
### 01-1 Unreal 소개
- 엔진 특징
### 01-2 프로젝트 생성
- 템플릿 선택
### 01-3 실행 확인
- Play In Editor

## 실습 체크리스트
- 기본 템플릿 프로젝트를 생성하고 PIE 실행을 확인한다.

## 본문
### 01-1 Unreal 소개
- Unreal은 고품질 3D 렌더링과 강력한 툴 체인을 제공하는 엔진입니다.
### 01-2 프로젝트 생성
- Blueprint 또는 C++ 기반 템플릿을 선택해 시작합니다.
### 01-3 실행 확인
- PIE(Play In Editor)로 즉시 테스트하며 반복 개발합니다.

약어 설명(PIE):
- **PIE**는 **Play In Editor**의 약자입니다.
- 에디터 내부에서 바로 플레이를 실행해 확인하는 기능입니다.
- 빠른 반복 테스트가 가능해 초기 개발 속도를 높입니다.

예시 코드(C++ 로그 확인):
```cpp
void AMyActor::BeginPlay()
{
    Super::BeginPlay();
    UE_LOG(LogTemp, Warning, TEXT("Hello Unreal"));
}
```

예상 결과:
```text
기본 맵에서 플레이어 이동 테스트 가능
```

코드 해석:
- `BeginPlay`는 게임 시작 시 호출됩니다.
- `UE_LOG`로 실행 시점 확인 로그를 출력할 수 있습니다.

연습문제:
1. 문제: Third Person 템플릿 생성
   - 입력: 프로젝트 이름
   - 출력: 실행 가능한 프로젝트
   - 조건: 프로젝트 경로 정리
2. 문제: PIE 실행
   - 입력: 없음
   - 출력: 에디터 내 실행 화면
   - 힌트: Play 버튼 사용

정답 포인트:
- 생성/실행 확인이 첫 단계
- 템플릿 선택은 학습 목적과 맞춰야 함
