# Chapter 06 입력과 캐릭터 이동

## 학습 목표
- 입력 매핑을 설정하고 캐릭터 이동에 연결할 수 있다.
- 카메라 기준 이동 개념을 이해한다.

## 세부 주제
### 06-1 입력 설정
- Axis/Action Mapping
### 06-2 이동 구현
- Forward/Right 입력
### 06-3 카메라 연동
- 시점 기준 방향

## 실습 체크리스트
- WASD 이동과 마우스 회전을 구현한다.

## 본문
### 06-1 입력 설정
- 입력 매핑으로 키를 기능 이름에 연결해 유지보수성을 높입니다.
### 06-2 이동 구현
- 입력 축 값을 받아 캐릭터 이동 함수를 호출합니다.
### 06-3 카메라 연동
- 카메라 회전 기준으로 이동 방향을 계산하면 조작감이 향상됩니다.

예시 코드:
```cpp
void AMyCharacter::MoveForward(float Value)
{
    AddMovementInput(GetActorForwardVector(), Value);
}
```

예상 결과:
```text
축 입력 값에 따라 캐릭터 전/후 이동
```

코드 해석:
- `AddMovementInput`에 방향 벡터와 입력 세기를 전달해 이동을 적용합니다.
- 입력 바인딩에서 이 함수를 연결하면 키 입력이 이동으로 변환됩니다.

연습문제:
1. 문제: 좌우 이동 함수 추가
   - 입력: A/D
   - 출력: 좌우 이동
   - 조건: 별도 함수 분리
2. 문제: 달리기 토글
   - 입력: Shift
   - 출력: 이동 속도 증가
   - 힌트: 최대 이동 속도 변수 변경

정답 포인트:
- 입력 매핑 이름을 명확히 정의
- 이동 함수를 축별로 분리하면 디버깅 쉬움
