# Chapter 09 애니메이션 상태머신

## 학습 목표
- 상태머신 기반 애니메이션 전환을 이해한다.
- 이동 속도/상태 변수로 전환 조건을 설정할 수 있다.

## 세부 주제
### 09-1 Anim Blueprint
- 애니메이션 제어 자산
### 09-2 상태머신 구성
- Idle/Run/Jump
### 09-3 전환 조건
- 속도/공중 여부

## 실습 체크리스트
- 캐릭터 이동 상태에 따라 애니메이션을 전환한다.

## 본문
### 09-1 Anim Blueprint
- Anim Blueprint는 스켈레탈 메쉬 애니메이션 로직을 담당합니다.
### 09-2 상태머신 구성
- 상태를 분리하면 전환 조건을 명확히 관리할 수 있습니다.
### 09-3 전환 조건
- 속도 값이나 점프 상태 등 측정 가능한 변수를 기준으로 전환합니다.

예시 코드(속도 변수 갱신):
```cpp
Speed = FVector::DotProduct(CharacterVelocity, CharacterForward);
```

예상 결과:
```text
정지/이동/점프 상태에 맞는 애니메이션 자동 전환
```

코드 해석:
- 이동 벡터 기반 속도 값을 애니메이션 변수로 전달합니다.
- 애님 상태머신은 이 변수로 전환 조건을 판정합니다.

연습문제:
1. 문제: 속도 기반 Idle-Run 전환
   - 입력: 이동 축 값
   - 출력: 상태 전환
   - 조건: 임계값 설정
2. 문제: 점프 착지 전환
   - 입력: 공중 여부
   - 출력: Jump -> Idle/Run 복귀
   - 힌트: IsFalling 변수 활용

정답 포인트:
- 상태 수를 최소로 시작하고 점진 확장
- 전환 조건은 겹치지 않게 설계
