## 실습 체크리스트
- 빈 프로젝트를 만들고 Play 버튼으로 실행 확인한다.

## 본문
### 01-1 Unity란?
- Unity는 게임 오브젝트와 컴포넌트를 조합해 게임을 만드는 엔진입니다.
- 2D/3D 프로젝트를 하나의 에디터에서 관리할 수 있습니다.

### 01-2 프로젝트 생성
- Unity Hub에서 버전과 템플릿(3D/2D)을 선택해 프로젝트를 만듭니다.
- 프로젝트 이름/경로를 명확히 정하면 협업 시 관리가 쉬워집니다.

### 01-3 실행 확인
- Play Mode는 현재 씬을 즉시 테스트하는 기능입니다.
- 실행 중 변경 사항은 기본적으로 저장되지 않으므로 주의가 필요합니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 엔진(engine) | 게임 실행/렌더링/입력 처리를 제공하는 기반 소프트웨어 |
| Play Mode | 에디터 안에서 게임을 미리 실행해보는 테스트 모드 |

예시 코드:
```csharp
using UnityEngine;

public class HelloUnity : MonoBehaviour
{
    void Start()
    {
        Debug.Log("Hello Unity");
    }
}
```

예상 결과:
```text
Console 창에 "Hello Unity" 로그 출력
```

코드 해석:
- `Start()`는 오브젝트 활성화 시점에 한 번 실행됩니다.
- `Debug.Log`는 실행 확인용 로그 출력에 사용합니다.
