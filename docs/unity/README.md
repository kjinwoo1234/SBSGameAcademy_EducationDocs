# Unity 자습 자료

## 권장 대상
- C# 기초를 학습했고 Unity로 게임 제작을 시작하려는 학습자
- 씬/오브젝트/컴포넌트 기반으로 실제 결과물을 만들어보고 싶은 학습자

## 선수지식
- 변수, 조건문, 함수 등 C# 기본 문법
- 파일/폴더 관리와 IDE 기본 사용

## 완주 후 역량
- 입력/물리/UI/애니메이션을 연결한 기본 게임 루프를 구현할 수 있다.
- 저장/로드와 빌드 배포까지 포함한 Unity 미니 프로젝트 파이프라인을 수행할 수 있다.
- 에디터 작업과 스크립트 작업의 역할을 구분해 개발 흐름을 설계할 수 있다.

- 리소스 관리: `Addressables` 기반 로딩/메모리 전략
