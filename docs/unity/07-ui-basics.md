# UI 기초 (Canvas, Text, Button)

Unity UI는 Canvas 아래에 배치됩니다.

## 기본 구성
- Canvas: UI 루트
- Text (TMP): 텍스트 표시
- Button: 클릭 입력

## 버튼 이벤트 연결
1. 버튼 선택
2. Inspector의 `On Click()`에 함수 연결
3. 대상 오브젝트 + 메서드 지정

## 실수 포인트
- 화면 해상도 대응: Canvas Scaler 설정 확인 (`Scale With Screen Size`)
