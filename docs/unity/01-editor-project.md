# Unity 에디터와 프로젝트 구조

Unity 프로젝트는 `Assets`, `Packages`, `ProjectSettings`를 중심으로 구성됩니다.

## 학습 목표
- Unity 에디터의 핵심 패널 역할을 구분할 수 있다.
- 프로젝트 기본 폴더 구조를 직접 만들 수 있다.
- 씬 저장/정리 습관을 적용할 수 있다.

## 핵심
- `Hierarchy`: 씬 오브젝트 목록
- `Scene`: 오브젝트 배치/편집
- `Inspector`: 선택 오브젝트 속성 편집
- `Project`: 에셋(스크립트, 이미지, 프리팹) 관리

## 초보자 체크포인트
- 폴더는 `Scripts`, `Prefabs`, `Materials`, `Scenes`처럼 기능별로 분리
- 씬 파일은 자주 저장 (`Ctrl + S`)
