# Chapter 10 도전! 프로그래밍 1

## 학습 목표
- 기초 문법을 문제 해결에 적용하는 연습을 한다.
- 반복/조건/재귀를 조합해 사고력을 높인다.

## 세부 주제
### 10-1 다양한 반복문 문제
- 누적합, 패턴 출력, 반복 최적화

### 10-2 다양한 조건문 문제
- 다중 조건 처리와 예외 케이스 점검

### 10-3 다양한 재귀함수 문제
- 재귀 호출 구조 설계와 종료 조건 검증

## 실습 체크리스트
- 각 주제에서 최소 2문제 이상 직접 구현해본다.

## 본문

### 10-1 다양한 반복문 문제
- 반복문 문제는 "반복 조건"과 "반복 변수 변화"를 정확히 잡는 연습입니다.
- 누적합, 배수 계산, 패턴 출력 문제를 통해 반복문 구조에 익숙해질 수 있습니다.
- 같은 문제를 `while`과 `for`로 각각 작성해보면 반복문 선택 기준이 명확해집니다.

### 10-2 다양한 조건문 문제
- 조건문 문제는 입력값에 따라 분기 흐름을 설계하는 연습입니다.
- 범위 조건, 다중 조건, 예외 입력 처리까지 포함해 작성해보는 것이 좋습니다.
- 구현 후에는 테스트 입력을 여러 개 넣어 예외 케이스를 반드시 점검합니다.

### 10-3 다양한 재귀함수 문제
- 재귀 문제는 "문제를 더 작은 같은 문제로 나누는 사고"가 핵심입니다.
- 먼저 종료 조건을 정하고, 그다음 재귀 호출 식을 작성하면 안정적으로 구현할 수 있습니다.
- 재귀가 어렵다면 호출 과정을 종이에 써보며 스택처럼 추적하는 연습이 큰 도움이 됩니다.

문제 풀이 순서:
| 단계 | 할 일 |
|---|---|
| 1 | 입력/출력 형식 먼저 확정 |
| 2 | 핵심 로직을 작은 단위 함수로 분리 |
| 3 | 기본 테스트 + 경계값 테스트 |
| 4 | 반복/조건/재귀 대체 풀이 비교 |

연습문제:
1. 반복문 문제 1 - 1부터 N까지 합
   - 문제: 정수 `N`을 입력받아 `1`부터 `N`까지의 합을 출력하세요.
   - 입력: 양의 정수 `N` 1개
   - 출력: 합계 1개
   - 조건: `N`은 1 이상 10000 이하
   - 예시: 입력 `5` -> 출력 `15`
2. 반복문 문제 2 - 구구단 출력
   - 문제: 단 `dan`을 입력받아 `dan x 1`부터 `dan x 9`까지 출력하세요.
   - 입력: 정수 `dan` 1개
   - 출력: 9줄
   - 조건: `dan`은 2 이상 9 이하
   - 예시: 입력 `3` -> 첫 줄 `3 x 1 = 3`
3. 반복문 문제 3 - 별 피라미드 출력
   - 문제: 높이 `h`를 입력받아 왼쪽 정렬 피라미드를 출력하세요.
   - 입력: 정수 `h` 1개
   - 출력: `h`줄
   - 조건: `h`는 1 이상 10 이하
   - 예시: 입력 `3` -> `*`, `**`, `***`
4. 조건문 문제 1 - 윤년 판별
   - 문제: 연도 `year`를 입력받아 윤년이면 `Leap`, 아니면 `Common`을 출력하세요.
   - 입력: 정수 `year` 1개
   - 출력: `Leap` 또는 `Common`
   - 조건: `year % 4 == 0`이고 `year % 100 != 0`이거나 `year % 400 == 0`이면 윤년
   - 예시: 입력 `2000` -> `Leap`, 입력 `1900` -> `Common`
5. 조건문 문제 2 - 세 수 중 최댓값
   - 문제: 정수 3개를 입력받아 가장 큰 값을 출력하세요.
   - 입력: 정수 `a b c`
   - 출력: 최댓값 1개
   - 예시: 입력 `4 9 2` -> 출력 `9`
6. 재귀 문제 1 - 팩토리얼
   - 문제: 정수 `N`을 입력받아 `N!`을 재귀 함수로 계산하세요.
   - 입력: 정수 `N` 1개
   - 출력: 팩토리얼 값
   - 조건: 종료 조건(기저 조건) 필수
   - 예시: 입력 `5` -> 출력 `120`
7. 재귀 문제 2 - 1부터 N까지 합
   - 문제: 정수 `N`을 입력받아 `1~N` 합을 재귀 함수로 계산하세요.
   - 입력: 정수 `N` 1개
   - 출력: 합계 1개
   - 조건: `N == 1`일 때 종료하도록 작성
   - 예시: 입력 `4` -> 출력 `10`
8. 재귀 문제 3(심화) - 문자열 뒤집기
   - 문제: 문자열을 입력받아 재귀 함수를 사용해 역순으로 출력하세요.
   - 입력: 문자열 1개(공백 없는 단어)
   - 출력: 뒤집힌 문자열
   - 힌트: 문자열 끝(널 문자)까지 재귀 호출 후, 복귀하면서 문자 출력

정답 예시(팩토리얼):
```c
#include <stdio.h>

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

int main(void) {
    int n = 5;
    printf("%d! = %d\n", n, factorial(n));
    return 0;
}
```

체크 기준:
- 입력 예외 처리(0, 음수, 큰 수)에 대한 대응이 있는가?
- 같은 문제를 다른 방식(반복/재귀)으로도 구현해봤는가?
- 함수 분리, 변수 이름, 출력 형식이 읽기 쉽게 정리되어 있는가?

정답 포인트:
- 반복문 문제는 반복 조건/증감식이 정확해야 합니다.
- 조건문 문제는 경계값(예: 59, 60, 89, 90) 테스트가 핵심입니다.
- 재귀 문제는 종료 조건을 가장 먼저 작성해야 합니다.
