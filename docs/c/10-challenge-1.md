# Chapter 10 도전! 프로그래밍 1

## 학습 목표
- 기초 문법을 문제 해결에 적용하는 연습을 한다.
- 반복/조건/재귀를 조합해 사고력을 높인다.

## 세부 주제
### 10-1 다양한 반복문 문제
- 누적합, 패턴 출력, 반복 최적화

### 10-2 다양한 조건문 문제
- 다중 조건 처리와 예외 케이스 점검

### 10-3 다양한 재귀함수 문제
- 재귀 호출 구조 설계와 종료 조건 검증

## 실습 체크리스트
- 각 주제에서 최소 2문제 이상 직접 구현해본다.

## 본문

### 10-1 다양한 반복문 문제

반복문 문제의 핵은 **언제 반복이 멈추는지**와 **매번 무엇이 바뀌는지**를 정확히 잡는 연습입니다. 누적합, 배수 계산, 별 패턴 출력처럼 규칙이 보이는 문제를 여러 개 풀면 구조에 익숙해집니다. 같은 문제를 `while`과 `for`로 각각 구현해 보면 “어떤 반복문을 고를지” 기준이 분명해집니다.

### 10-2 다양한 조건문 문제

조건문 문제는 입력값에 따라 **분기**를 어떻게 나눌지 설계하는 연습입니다. 범위 조건, 여러 조건을 동시에 보는 경우, 이상한 입력이 들어온 경우까지 생각해 두면 좋습니다. 구현이 끝나면 테스트 입력을 여러 개 넣어 **경계값**이 잘 통과하는지 반드시 확인합니다.

### 10-3 다양한 재귀함수 문제

재귀는 “큰 문제를 **같은 모양의 작은 문제**로 쪼갠다”는 사고가 중심입니다. 먼저 **종료 조건**을 적고, 그다음에 재귀 호출 식을 쓰면 구현이 훨씬 안정적입니다. 머릿속이 복잡하면 호출을 종이에 쌓아 올리듯 적어 보며 스택을 따라가 보는 것도 큰 도움이 됩니다.

문제를 풀 때 권장하는 순서는 아래 표와 같습니다.

| 단계 | 할 일 |
|---|---|
| 1 | 입력/출력 형식 먼저 확정 |
| 2 | 핵심 로직을 작은 단위 함수로 분리 |
| 3 | 기본 테스트 + 경계값 테스트 |
| 4 | 반복/조건/재귀 대체 풀이 비교 |

### 연습문제

**문제 1 (반복문)**
- 문제: 정수 `N`을 입력받아 1부터 `N`까지의 합을 출력하세요.
- 입력: 양의 정수 `N` 1개 (`1 <= N <= 10000`)
- 출력: 합계 1개 (예: 입력 `5` -> 출력 `15`)
- 조건(힌트): 누적합 변수를 두고 반복문으로 `1`부터 `N`까지 더하세요.

**문제 2 (반복문)**
- 문제: 단 `dan`을 입력받아 `dan x 1`부터 `dan x 9`까지 출력하세요.
- 입력: 정수 `dan` 1개 (`2 <= dan <= 9`)
- 출력: 구구단 9줄
- 조건(힌트): 반복문 안에서 곱셈 결과를 한 줄씩 출력하세요.

**문제 3 (반복문)**
- 문제: 높이 `h`를 입력받아 왼쪽 정렬 별 피라미드를 출력하세요.
- 입력: 정수 `h` 1개 (`1 <= h <= 10`)
- 출력: `h`줄 별 패턴 (예: `h=3` -> `*`, `**`, `***`)
- 조건(힌트): 바깥 반복문은 줄 수, 안쪽 반복문은 별 개수를 담당하도록 구성하세요.

**문제 4 (조건문)**
- 문제: 연도 `year`를 입력받아 윤년이면 `Leap`, 아니면 `Common`을 출력하세요.
- 입력: 정수 연도 1개
- 출력: `Leap` 또는 `Common`
- 조건(힌트): `year % 4 == 0 && year % 100 != 0` 또는 `year % 400 == 0` 조건을 사용하세요.

**문제 5 (조건문)**
- 문제: 정수 세 개를 입력받아 가장 큰 값을 출력하세요.
- 입력: 정수 3개
- 출력: 최댓값 1개 (예: `4 9 2` -> `9`)
- 조건(힌트): `if-else if-else` 또는 비교 갱신 방식 중 하나를 사용하세요.

**문제 6 (재귀)**
- 문제: 정수 `N`을 입력받아 `N!`을 재귀 함수로 계산하세요.
- 입력: 정수 `N` 1개
- 출력: `N!` 값 1개 (예: `5` -> `120`)
- 조건(힌트): 종료 조건(기저 조건)을 먼저 작성하세요.

**문제 7 (재귀)**
- 문제: 정수 `N`을 입력받아 1부터 `N`까지의 합을 재귀 함수로 계산하세요.
- 입력: 정수 `N` 1개
- 출력: `1 + 2 + ... + N` 결과값 (예: `4` -> `10`)
- 조건(힌트): `N == 1` 또는 `N <= 1`을 종료 조건으로 두세요.

**문제 8 (재귀, 심화)**
- 문제: 공백 없는 단어 하나를 입력받아 재귀로 역순 출력하세요.
- 입력: 문자열 1개
- 출력: 입력 문자열의 역순
- 조건(힌트): 문자열 끝(널 문자)까지 재귀 호출 후, 복귀하면서 문자를 출력하세요.

### 정답 예시 (팩토리얼)

```c
#include <stdio.h>

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

int main(void) {
    int n = 5;
    printf("%d! = %d\n", n, factorial(n));
    return 0;
}
```

### 스스로 점검할 때 볼 것

입력이 0, 음수, 아주 큰 수일 때 어떻게 동작하는지, 같은 문제를 반복문과 재귀 중 둘 다로 풀 수 있는지, 함수 이름·출력 형식이 읽기 쉬운지를 점검해 보세요.

### 정답 포인트

반복문 문제는 조건과 증감이 정확해야 하고, 조건문 문제는 경계값(예: 59, 60, 89, 90) 테스트가 중요합니다. 재귀 문제는 **종료 조건을 가장 먼저** 고정하는 것이 핵심입니다.
