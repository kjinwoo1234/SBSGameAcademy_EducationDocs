# C 함수

함수는 코드를 기능 단위로 나누어 재사용성과 가독성을 높입니다.  
특히 규모가 커질수록 함수 분해 능력이 코드 품질을 결정합니다.

## 1) 함수의 기본 구조
함수는 반환형, 함수명, 매개변수, 함수 본문으로 구성됩니다.

```c
int Add(int a, int b) {
    return a + b;
}
```

## 2) 선언(프로토타입)과 정의
메인 함수 아래에 정의할 경우, 위쪽에 선언이 필요합니다.

```c
#include <stdio.h>

int Add(int a, int b); // 선언

int main(void) {
    printf("%d\n", Add(3, 5));
    return 0;
}

int Add(int a, int b) { // 정의
    return a + b;
}
```

## 3) 값 전달(Call by Value)
C 함수는 기본적으로 값을 복사해 전달합니다.  
원본을 변경하려면 포인터를 매개변수로 받아야 합니다.

```c
void Increase(int* x) {
    (*x)++;
}
```

## 4) 함수 분리 원칙
- 함수 하나는 한 가지 책임만 갖게 설계
- 함수명은 동작이 드러나게 작성 (`CalcAverage`, `PrintMenu` 등)

## 5) 연습 문제
1. 두 수 중 큰 값을 반환하는 함수 작성
2. 배열 평균을 계산하는 함수 작성
3. 포인터를 사용해 두 변수 값을 swap하는 함수 작성
