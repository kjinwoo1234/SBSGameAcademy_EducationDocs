# Chapter 04 데이터 표현방식의 이해

## 학습 목표
- 컴퓨터가 데이터를 비트 단위로 표현하는 원리를 이해한다.
- 정수/실수 및 비트 연산의 기초를 익힌다.

## 세부 주제
### 04-1 컴퓨터의 데이터 표현
- 2진수, 부호 표현, 메모리 단위

### 04-2 정수/실수 표현 방식
- 정수형 범위와 부동소수점 개념

### 04-3 비트 연산자
- AND, OR, XOR, SHIFT 활용

## 실습 체크리스트
- 비트 마스크를 이용해 특정 비트 ON/OFF 코드를 작성한다.

## 본문

### 04-1 컴퓨터의 데이터 표현
- 컴퓨터는 모든 데이터를 0(전기가 흐르지 않음)과 1(전기가 흐르고 있음)로 저장합니다. 이 단위를 비트라고 합니다.
- 8비트가 1바이트이며, 자료형마다 사용하는 바이트 수가 다릅니다.
- 결국 문자, 숫자, 이미지도 내부에서는 비트 조합으로 저장됩니다.

용량 단위 기초:
- 1 bit: 0 또는 1 한 자리
- 8 bit = 1 byte
- 1024 byte = 1 KB
- 1024 KB = 1 MB

자주 쓰는 자료형 용량(일반적인 Windows MSVC 기준):
- `char`: 1바이트
- `int`: 4바이트
- `float`: 4바이트
- `double`: 8바이트

### 04-2 정수/실수 표현 방식
- 정수는 부호와 크기를 정해진 비트 수 안에 저장합니다.
- 실수는 소수점을 포함한 값을 저장하기 위해 별도의 표현 규칙(부동소수점)을 사용합니다.
- 실수 계산에서는 작은 오차가 발생할 수 있으므로, 결과 비교 시 주의가 필요합니다.

부동소수점 오차가 생기는 이유(초간단):
- 10진수의 일부 값(예: 0.1)은 2진수로 정확히 표현되지 않습니다.
- 그래서 메모리에 "가장 가까운 값"으로 저장되고, 계산 과정에서 미세한 오차가 보일 수 있습니다.

오차 예시:
```c
#include <stdio.h>

int main(void) {
    double x = 0.1 + 0.2;
    printf("%.17f\n", x); // 0.30000000000000004 처럼 보일 수 있음
    return 0;
}
```

### 04-3 비트 연산자
- 비트 연산은 숫자를 비트 단위로 직접 조작할 수 있는 기능입니다.
- `&`, `|`, `^`, `~`, `<<`, `>>`를 이용해 특정 비트를 켜거나 끄는 작업을 할 수 있습니다.
- 게임/임베디드/시스템 코드에서 상태 플래그를 효율적으로 관리할 때 자주 사용됩니다.

비트 연산자 기능 요약:
- `&` (AND): 두 비트가 모두 1일 때만 1
- `|` (OR): 둘 중 하나라도 1이면 1
- `^` (XOR): 두 비트가 다를 때 1
- `~` (NOT): 0과 1을 반전
- `<<` (왼쪽 시프트): 비트를 왼쪽으로 이동(보통 값이 2배씩 증가)
- `>>` (오른쪽 시프트): 비트를 오른쪽으로 이동(보통 값이 2로 나눠짐)

비트 연산 예시:
```c
#include <stdio.h>

int main(void) {
    unsigned char flags = 0; // 00000000

    flags |= (1 << 0); // bit0 ON
    flags |= (1 << 2); // bit2 ON -> 00000101
    printf("flags: %u\n", flags); // 5

    flags &= ~(1 << 0); // bit0 OFF -> 00000100
    printf("flags: %u\n", flags); // 4

    printf("bit2 is on? %d\n", (flags & (1 << 2)) != 0); // 1
    return 0;
}
```

연습문제:
1. `num = 12`일 때 `num << 1`, `num >> 1` 결과를 구하세요.
2. 정수의 3번째 비트를 ON으로 만드는 코드를 작성하세요.
3. 정수의 2번째 비트가 켜져 있는지 검사하는 코드를 작성하세요.

정답:
1. `24`, `6`
2. `num |= (1 << 2);`
3. `(num & (1 << 1)) != 0`
