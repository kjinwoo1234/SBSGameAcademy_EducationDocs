# C 연산자

연산자는 값을 계산하고 비교하며, 조건을 조합하는 핵심 도구입니다.  
실무에서는 "연산 결과 타입"과 "우선순위"를 정확히 이해하는 것이 중요합니다.

## 1) 산술 연산자
- `+`, `-`, `*`, `/`, `%`
- `%`는 정수에서만 사용

```c
int x = 10, y = 3;
printf("x+y = %d\n", x + y);   // 13
printf("x/y = %d\n", x / y);   // 3
printf("x%%y = %d\n", x % y);  // 1
```

## 2) 증감 연산자
- `++a`: 먼저 증가 후 사용 (전위)
- `a++`: 먼저 사용 후 증가 (후위)

```c
int a = 5;
int b = ++a; // a=6, b=6
int c = a++; // c=6, a=7
```

## 3) 비교 연산자
- `==`, `!=`, `>`, `<`, `>=`, `<=`
- 결과는 참(1) 또는 거짓(0)

```c
int score = 85;
printf("%d\n", score >= 80); // 1
printf("%d\n", score == 100); // 0
```

## 4) 논리 연산자
- `&&`: 모두 참일 때 참
- `||`: 하나라도 참이면 참
- `!`: 참/거짓 반전

```c
int age = 20;
int pass = (age >= 19) && (age < 30);
printf("%d\n", pass); // 1
```

## 5) 대입/복합 대입
- `=`, `+=`, `-=`, `*=`, `/=`, `%=`

```c
int money = 1000;
money += 500; // 1500
money -= 200; // 1300
```

## 6) 연산자 우선순위
산술 > 비교 > 논리 순으로 계산됩니다.  
헷갈릴 때는 괄호를 적극적으로 사용하세요.

## 7) 연습 문제
1. 세 정수의 합, 평균, 최댓값을 계산하세요.
2. 점수가 60 이상이고 출석이 80 이상일 때 합격 처리하세요.
3. 전위/후위 증감 결과를 출력해 차이를 확인하세요.
