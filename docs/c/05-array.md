# C 배열

배열은 동일한 자료형 데이터를 연속된 메모리 공간에 저장하는 구조입니다.  
반복문과 함께 사용하면 대량 데이터를 효율적으로 처리할 수 있습니다.

## 1) 1차원 배열
```c
int scores[5] = {80, 90, 70, 85, 100};
printf("%d\n", scores[0]); // 첫 번째 요소
```

- 인덱스는 `0`부터 시작
- 유효 범위: `0` ~ `크기-1`

## 2) 배열 순회
```c
int sum = 0;
for (int i = 0; i < 5; i++) {
    sum += scores[i];
}
printf("총합: %d\n", sum);
```

## 3) 2차원 배열
행/열 구조의 데이터를 표현할 때 사용합니다.

```c
int board[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
printf("%d\n", board[1][2]); // 6
```

## 4) 배열 초기화 규칙
```c
int a[5] = {1, 2};   // 나머지는 0으로 초기화
int b[] = {3, 4, 5}; // 크기 자동 추론
```

## 5) 자주 하는 실수
- 범위를 넘는 인덱스 접근 (`scores[5]` 등)
- 반복문 종료 조건을 잘못 작성 (`<=` vs `<`)
