## 본문

### 22-1 구조체 기초

구조체(structure)는 서로 다른 타입의 멤버를 **한 덩어리 타입**으로 묶는 사용자 정의 자료형입니다. 변수에 직접 붙일 때는 점(`.`)으로 멤버에 접근하고, 구조체를 가리키는 포인터가 있으면 화살표(`->`)로 접근합니다.

### 22-2 구조체/배열/포인터

구조체 **배열**은 같은 모양의 객체를 여러 개 나열해 관리할 때 편합니다. 구조체 **포인터**로 넘기면 함수 호출 시 전체 복사를 피할 수 있어, 덩어리가 커질수록 유리한 경우가 많습니다.

구조체를 값으로 넘기면 호출 시 **내용 전체가 복사**됩니다. 구조체가 크면 복사 비용이 커질 수 있어, 포인터로는 “주소 하나”만 넘기는 방식이 부담이 적을 때가 많습니다. 아래 예시에서는 값 복사 없이 `main` 안에서만 멤버를 읽습니다. 함수로 넘길 때는 `struct Student s`처럼 값으로 넘길지, `const struct Student *p`처럼 주소로 넘길지 설계에 맞게 고르면 됩니다.

| 상황 | 접근 연산자 | 예시 |
|---|---|---|
| 구조체 변수 | `.` | `s.id` |
| 구조체 포인터 | `->` | `p->id` |

```c
#include <stdio.h>

struct Student {
    int id;
    char grade;
    int score;
};

int main(void) {
    struct Student s[2] = {{1, 'A', 95}, {2, 'B', 82}};
    for (int i = 0; i < 2; i++) {
        printf("%d %c %d\n", s[i].id, s[i].grade, s[i].score);
    }
    return 0;
}
```

**예상 출력**

```text
1 A 95
2 B 82
```
