# Chapter 11 1차원 배열

## 학습 목표
- 배열의 메모리 연속성 개념을 이해한다.
- 문자열과 문자 배열의 관계를 익힌다.

## 세부 주제
### 11-1 배열 선언/초기화
- 인덱스 접근, 반복문 순회, 범위 주의

### 11-2 배열 기반 문자열
- 널 문자와 문자열 처리 기초

## 실습 체크리스트
- 배열 통계를 계산하는 프로그램(최대/최소/평균)을 작성한다.

## 본문

### 11-1 배열 선언/초기화
- 배열은 같은 타입 데이터를 연속된 메모리 공간에 저장합니다.
- 인덱스는 `0`부터 시작하며, 마지막 인덱스는 `크기 - 1`입니다.
- 범위를 벗어난 접근은 런타임 오류 원인이 되므로 가장 주의해야 합니다.

### 11-2 배열 기반 문자열
- 문자열은 `char` 배열이며 끝에 `\0`(널 문자)가 들어갑니다.
- 문자열 길이와 배열 크기를 구분해서 생각해야 안전한 코드를 작성할 수 있습니다.

예시 코드:
```c
#include <stdio.h>

int main(void) {
    int nums[5] = {10, 20, 30, 40, 50};
    int i, sum = 0;

    for (i = 0; i < 5; i++) sum += nums[i];
    printf("합: %d\n", sum);

    char name[10] = "Kim";
    printf("이름: %s\n", name);
    return 0;
}
```

연습문제:
1. 정수 5개를 배열에 저장하고 최댓값을 출력하세요.
2. 정수 5개의 평균을 계산해 출력하세요.
3. 문자열을 입력받아 길이를 출력하세요. (`strlen` 사용 가능)

정답 포인트:
- 반복문 인덱스 범위: `i < 배열크기`
- 평균 계산 시 실수형 캐스팅 고려
