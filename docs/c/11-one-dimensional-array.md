## 본문

### 11-1 배열 선언/초기화
- 배열은 같은 타입 데이터를 연속된 메모리 공간에 저장합니다.
- 인덱스(index)는 배열 안에서 각 원소의 "번호(위치)"를 의미합니다.
- 인덱스는 `0`부터 시작하며, 마지막 인덱스는 `크기 - 1`입니다.
- 범위를 벗어난 접근은 런타임 오류 원인이 되므로 가장 주의해야 합니다.

인덱스 예시:
- `int nums[5] = {10, 20, 30, 40, 50};`
- `nums[0]`은 `10`, `nums[1]`은 `20`, `nums[4]`는 `50`입니다.
- 이 배열에서 유효한 인덱스는 `0, 1, 2, 3, 4`입니다.

용어 설명:
- 런타임(runtime)은 프로그램이 실제로 실행되는 시간/환경을 말합니다.
- 즉, 런타임 오류는 컴파일은 되었지만 실행 중에 발생하는 오류입니다.

### 11-2 배열 기반 문자열
- 문자열은 `char` 배열이며 끝에 `\0`(널 문자)가 들어갑니다.
- 문자열 길이와 배열 크기를 구분해서 생각해야 안전한 코드를 작성할 수 있습니다.

널 문자(`\0`) 설명:
- 널 문자는 "문자열의 끝"을 표시하는 특수 문자입니다.
- C에서 문자열 함수(`printf("%s")`, `strlen` 등)는 `\0`을 만날 때까지를 문자열로 처리합니다.
- 예를 들어 `char name[4] = "Kim";`은 내부적으로 `K`, `i`, `m`, `\0`이 저장됩니다.
- 배열 크기를 문자열 길이보다 1 크게 잡는 이유가 바로 이 `\0`을 저장하기 위해서입니다.

배열/문자열 구분:
| 항목 | 정수 배열 | 문자열 배열 |
|---|---|---|
| 타입 | `int[]` | `char[]` |
| 끝 표시 | 없음 | `\0` 있음 |
| 출력 방식 | 반복문 순회 | `%s` 가능 |

예시 코드:
```c
#include <stdio.h>

int main(void) {
    int nums[5] = {10, 20, 30, 40, 50};
    int i, sum = 0;

    for (i = 0; i < 5; i++) sum += nums[i];
    printf("합: %d\n", sum);

    char name[10] = "Kim";
    printf("이름: %s\n", name);
    return 0;
}
```

예상 출력:
```text
합: 150
이름: Kim
```

   - 출력: 평균 1개

   - 문제: 문자열 1개를 입력받아 길이를 출력하세요.
   - 입력: 공백 없는 문자열 1개
   - 출력: 문자열 길이
   - 힌트: `strlen` 사용 가능

정답 포인트:
