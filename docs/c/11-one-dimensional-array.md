## 본문

### 11-1 배열 선언/초기화

배열은 **같은 타입**의 데이터를 메모리에 **연속해서** 놓는 방법입니다. **인덱스(index)**는 배열 안에서 각 칸의 번호(위치)이고, C에서는 **0부터** 시작합니다. 크기가 `N`이면 유효한 인덱스는 `0`부터 `N - 1`까지입니다. 이 범위를 넘어가면 **런타임(runtime, 실행 중)** 오류나 정의되지 않은 동작으로 이어질 수 있으므로 가장 조심해야 할 부분입니다.

예를 들어 `int nums[5] = {10, 20, 30, 40, 50};`이면 `nums[0]`은 10, `nums[4]`는 50이고, 유효한 인덱스는 `0, 1, 2, 3, 4`입니다.

### 11-2 배열 기반 문자열

C에서 흔히 말하는 문자열은 `char` 배열이며, 끝에는 **널 문자** `\0`이 들어갑니다. “문자열이 차지하는 글자 수”와 “배열을 몇 칸 잡았는지”는 다른 개념이므로, 배열 크기는 보통 문자 개수보다 **최소 1칸 더** 잡아 `\0`을 둘 자리를 남깁니다.

`\0`은 “여기까지가 문자열”이라고 표시하는 특수 문자입니다. `printf("%s", ...)`나 `strlen` 같은 함수는 `\0`을 만날 때까지를 한 덩어리의 문자열로 봅니다. 예를 들어 `char name[4] = "Kim";`은 `K`, `i`, `m`, `\0` 네 칸을 쓰는 그림으로 이해할 수 있습니다.

| 항목 | 정수 배열 | 문자열 배열 |
|---|---|---|
| 타입 | `int[]` | `char[]` |
| 끝 표시 | 없음 | `\0` 있음 |
| 출력 방식 | 반복문 순회 | `%s` 가능 |

```c
#include <stdio.h>

int main(void) {
    int nums[5] = {10, 20, 30, 40, 50};
    int i, sum = 0;

    for (i = 0; i < 5; i++) sum += nums[i];
    printf("합: %d\n", sum);

    char name[10] = "Kim";
    printf("이름: %s\n", name);
    return 0;
}
```

**예상 출력**

```text
합: 150
이름: Kim
```

### 연습문제

**문제 1 — 배열 최댓값**  
정수 다섯 개를 배열에 저장한 뒤 최댓값을 출력하세요. 입력은 정수 다섯 개, 출력은 최댓값 하나입니다. 반복문으로 배열 전체를 순회합니다.

**문제 2 — 배열 평균**  
정수 다섯 개의 평균을 소수 둘째 자리까지 출력하세요. 정수 나눗셈만 하면 의도와 다른 결과가 나올 수 있으므로 **실수형 캐스팅**을 고려합니다.

**문제 3 — 문자열 길이**  
공백 없는 문자열 하나를 입력받아 길이를 출력하세요. `strlen` 사용이 가능합니다.

### 정답 포인트

반복문 인덱스는 `i < 배열크기`처럼 범위를 엄격히 맞추고, 평균은 `(double)합 / 개수`처럼 실수 나눗셈이 되도록 조정합니다.
