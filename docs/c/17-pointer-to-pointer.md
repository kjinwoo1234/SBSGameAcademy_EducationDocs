## 본문

### 17-1 이중 포인터

이중 포인터는 `type **pp`처럼 쓰며, “**포인터를 가리키는 포인터**”입니다. 값에 한 번 더 간접적으로 손대야 할 때, 즉 접근 단계가 한 층 더 필요할 때 등장합니다.

### 17-2 다중 포인터의 필요성

함수 안에서 **포인터 변수 자체**를 바꾸고 싶을 때 이중 포인터가 필요합니다. 예를 들어 문자열을 가리키는 포인터를 다른 문자열로 바꾸거나, 동적 메모리 블록을 가리키는 포인터를 재할당한 뒤 갱신할 때 자주 보입니다.

| 표현 | 의미 |
|---|---|
| `p` | 실제 데이터 주소 |
| `*pp` | 포인터 변수(`p`) |
| `**pp` | 실제 데이터 값 |

```c
#include <stdio.h>

void change_text(const char **p) {
    *p = "Changed";
}

int main(void) {
    const char *msg = "Original";
    change_text(&msg);
    printf("%s\n", msg);
    return 0;
}
```

**예상 출력**

```text
Changed
```

### 연습문제

**문제 1**
- 문제: `int *p`를 함수 안에서 다른 배열의 시작 주소로 바꾸세요.
- 입력: 정수 배열 2개, 포인터 변수 `p`
- 출력: 호출 전/후에 포인터가 가리키는 값
- 조건(힌트): 함수 인자는 `int **pp`를 사용하고 `*pp`를 새 주소로 바꾸세요.

**문제 2**
- 문제: 문자열 포인터를 함수에서 `"Before"`에서 `"After"`로 바꾸세요.
- 입력: 문자열 포인터 변수 1개
- 출력: 변경 전 문자열과 변경 후 문자열
- 조건(힌트): 함수 호출 시 `&포인터변수`를 넘기고, 함수 내부에서 `*p`를 변경하세요.

### 정답 포인트

포인터 **변수 자체**를 바꾸려면 `type **`가 필요하고, 호출할 때는 그 변수의 주소인 `&포인터변수`를 넘기면 됩니다.
