# Chapter 07 반복문

## 학습 목표
- 반복문의 종류와 사용 기준을 이해한다.
- 반복 제어 변수를 올바르게 다룰 수 있다.

## 세부 주제
### 07-1 `while`
- 조건 기반 반복의 기본 패턴

### 07-2 `do-while`
- 최소 1회 실행이 필요한 경우

### 07-3 `for`
- 카운트 기반 반복과 중첩 반복

## 실습 체크리스트
- `while`과 `for`로 동일 문제를 각각 구현해 비교한다.

## 본문

### 07-1 `while`
- `while`은 조건이 참인 동안 계속 반복합니다.
- 반복 횟수가 정해지지 않았거나, 특정 조건을 만족할 때까지 반복할 때 적합합니다.
- 무한 루프를 방지하려면 반복문 안에서 조건이 바뀌도록 코드를 작성해야 합니다.

### 07-2 `do-while`
- `do-while`은 조건 검사 전에 본문을 먼저 실행합니다.
- 그래서 조건이 거짓이어도 최소 1번은 실행됩니다.
- 메뉴를 한 번 보여주고 입력받는 구조처럼 "최소 1회 실행"이 필요한 상황에 유용합니다.

### 07-3 `for`
- `for`는 초기값, 조건식, 증감식을 한 줄에 모아 작성하는 반복문입니다.
- 반복 횟수가 명확한 문제(예: 1부터 100까지 합)에 가장 많이 사용합니다.
- 중첩 `for`를 사용하면 별 찍기, 2차원 배열 순회 같은 문제를 해결할 수 있습니다.

반복문 비교:
| 반복문 | 실행 시점 | 적합한 상황 | 주의점 |
|---|---|---|---|
| `while` | 조건 확인 후 실행 | 종료 조건이 이벤트/입력에 따라 바뀔 때 | 조건 갱신 누락 시 무한 루프 |
| `do-while` | 본문 1회 실행 후 조건 확인 | 최소 1회 실행이 필요한 메뉴/입력 루프 | 끝에 세미콜론(`;`) 필요 |
| `for` | 초기화 -> 조건 -> 본문 -> 증감 | 반복 횟수가 명확할 때 | 인덱스 범위 실수 주의 |

예시 코드:
```c
#include <stdio.h>

int main(void) {
    int i, sum = 0;

    for (i = 1; i <= 5; i++) {
        sum += i;
    }
    printf("1~5 합: %d\n", sum);

    i = 5;
    while (i > 0) {
        printf("%d ", i);
        i--;
    }
    printf("\n");
    return 0;
}
```

코드 해석:
- `for (i = 1; i <= 5; i++)`  
  - `i = 1`: 시작값  
  - `i <= 5`: 반복 조건  
  - `i++`: 한 번 반복 후 증가
- `sum += i;`로 1부터 5까지 누적합을 계산합니다.
- `while (i > 0)`는 조건이 참인 동안 숫자를 거꾸로 출력합니다.
- `i--`가 없으면 조건이 바뀌지 않아 반복문이 끝나지 않을 수 있습니다.

예상 출력:
```text
1~5 합: 15
5 4 3 2 1
```

`do-while` 예시:
```c
#include <stdio.h>

int main(void) {
    int menu = 0;

    do {
        printf("메뉴: 1. 시작  2. 종료\n");
        printf("입력: ");
        scanf("%d", &menu);
    } while (menu != 2);

    printf("프로그램 종료\n");
    return 0;
}
```

연습문제:

3. `do-while`로 메뉴를 1번 이상 출력하는 코드를 작성하세요.

정답 예시:
```c
for (int i = 2; i <= 10; i += 2) printf("%d ", i);
int n = 10;
while (n >= 1) { printf("%d ", n); n--; }

int menu = 0;
do {
    printf("메뉴 출력\n");
    scanf("%d", &menu);
} while (menu != 2);
```
