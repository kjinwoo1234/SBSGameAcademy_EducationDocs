## 본문

### 18-1 2차원 배열 이름의 포인터형

`int arr[2][3]`에서 배열 이름 `arr`의 타입은 대략 **`int (*)[3]`**에 가깝습니다. 한 줄로 말하면 “**int 세 개짜리 한 줄(행)**을 가리키는 포인터”입니다. 그래서 `arr`는 첫 행 전체를 가리키는 그림으로 읽는 경우가 많습니다.

### 18-2 2차원 배열 이름의 특성과 주의사항

2차원 배열을 함수 인자로 넘길 때는 **열의 크기**를 컴파일러가 한 줄의 폭을 계산하는 데 쓰므로, 매개변수에 **열 크기를 밝혀** 주는 형태가 흔합니다. 예를 들어 `void print_map(int arr[][3], int row)`처럼 씁니다.

| 선언 | 의미 |
|---|---|
| `int arr[][3]` | 열 크기 3 고정 |
| `int (*arr)[3]` | `int[3]` 배열을 가리키는 포인터 |

```c
#include <stdio.h>

void print_sum(int arr[][3], int row) {
    int sum = 0;
    for (int r = 0; r < row; r++)
        for (int c = 0; c < 3; c++)
            sum += arr[r][c];
    printf("sum: %d\n", sum);
}

int main(void) {
    int data[2][3] = {{1,2,3},{4,5,6}};
    print_sum(data, 2);
    return 0;
}
```

**예상 출력**

```text
sum: 21
```

### 연습문제

**문제 1**
- 문제: 2차원 배열과 행 수를 받아 최댓값을 반환하는 함수를 작성하세요.
- 입력: 2차원 배열, 행 수
- 출력: 배열 최댓값 1개
- 조건(힌트): 함수 매개변수 선언에 열 크기를 명시하세요.

**문제 2**
- 문제: 2차원 배열의 각 행의 합을 한 줄씩 출력하세요.
- 입력: 2차원 배열 원소들
- 출력: 행별 합계 목록
- 조건(힌트): 바깥 반복문은 행, 안쪽 반복문은 열을 담당하게 구성하세요.

### 정답 포인트

함수 매개변수에서 열 크기를 고정해 두고, 중첩 반복문으로 행·열을 빠짐없이 도는 것이 핵심입니다.
