## 본문

### 12-1 포인터 개념
- 주소(address)는 메모리 칸 하나하나를 구분하기 위한 "위치 번호"입니다.
- 포인터는 일반 값이 아니라, 이 "위치 번호(주소)"를 저장하는 변수입니다.
- `int *p`는 `int`형 데이터가 있는 주소를 저장하는 포인터라는 뜻입니다.
- 포인터 타입과 대상 변수 타입은 맞춰서 사용하는 것이 원칙입니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 메모리(memory) | 프로그램 실행 중 데이터가 저장되는 공간 |
| 주소(address) | 메모리 안에서 데이터가 저장된 위치 번호 |
| 포인터(pointer) | 주소를 저장하는 변수 |

왜 주소가 필요한가:
| 상황 | 주소가 필요한 이유 |
|---|---|
| 함수에서 원본 값 변경 | 값 복사본이 아니라 원본 위치를 알아야 직접 수정 가능 |
| 배열/문자열 순회 | 첫 위치 주소를 기준으로 다음 원소에 접근 가능 |
| 동적 메모리 사용 | 실행 중 확보한 메모리 위치를 저장해 재사용/해제 가능 |

### 12-2 `&`, `*` 연산자
- `&`는 변수의 주소를 얻을 때 사용합니다.
- `*`는 포인터가 가리키는 주소의 실제 값을 읽거나 수정할 때 사용합니다.
- 주소는 `printf`의 `%p` 형식 지정자로 출력해서 확인할 수 있습니다.
- `%p`로 출력할 때는 `(void *)`로 형변환해서 출력하는 습관을 권장합니다.

연산자 의미:
| 기호 | 의미 | 예시 |
|---|---|---|
| `&` | 주소 얻기 | `p = &num;` |
| `*` | 역참조(값 읽기/수정) | `*p = 20;` |

예시 코드:
```c
#include <stdio.h>

int main(void) {
    int num = 10;
    int *p = &num;

    printf("num: %d\n", num);
    printf("&num: %p\n", (void *)&num);
    printf("p   : %p\n", (void *)p);
    printf("&num == p: %d\n", (&num == p));

    *p = 20;
    printf("num after *p = 20: %d\n", num);
    return 0;
}
```

예상 출력(주소는 실행마다 다름):
```text
num: 10
&num: 0x...
p   : 0x...
&num == p: 1
num after *p = 20: 20
```

연습문제:
1. 문제 1 - 포인터로 값 변경
   - 문제: 정수 변수 `num`을 선언하고 포인터를 이용해 값을 2배로 바꾸세요.
   - 입력: 없음(코드 내부 초기값 사용)
   - 출력: 변경 전/후 값
   - 조건: 값 변경은 반드시 `*포인터`로 수행
2. 문제 2 - 포인터로 두 수 합 계산
   - 문제: 정수 변수 2개를 선언한 뒤 포인터를 사용해 합계를 출력하세요.
   - 입력: 없음(코드 내부 초기값 사용)
   - 출력: 합계 1개
   - 조건: 직접 변수명을 더하지 말고 포인터 역참조로 계산

정답 포인트:
- 포인터 선언: `자료형 *이름`
- 값 변경: `*포인터 = 새값`
