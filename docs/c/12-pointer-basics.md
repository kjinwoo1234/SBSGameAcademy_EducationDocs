## 본문

### 12-1 포인터 개념
- 포인터는 "값"이 아니라 "주소"를 저장하는 변수입니다.
- `int* p`는 `int`형 데이터의 주소를 저장하는 포인터라는 뜻입니다.
- 포인터 타입과 대상 변수 타입은 맞춰서 사용하는 것이 원칙입니다.

### 12-2 `&`, `*` 연산자
- `&`는 변수의 주소를 얻을 때 사용합니다.
- `*`는 포인터가 가리키는 주소의 실제 값을 읽거나 수정할 때 사용합니다.

예시 코드:
```c
#include <stdio.h>

int main(void) {
    int num = 10;
    int *p = &num;

    printf("num: %d\n", num);
    printf("address: %p\n", (void*)p);

    *p = 20;
    printf("num after *p = 20: %d\n", num);
    return 0;
}
```

정답 포인트:
