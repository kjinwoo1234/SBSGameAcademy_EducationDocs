# Chapter 16 다차원 배열

## 학습 목표
- 2차원/3차원 배열의 선언과 접근을 익힌다.
- 행과 열 기준 순회의 차이를 이해한다.

## 세부 주제
### 16-1 다차원 배열 활용
- 표 형태 데이터 저장과 탐색

### 16-2 3차원 배열
- 좌표 기반 데이터 모델링

## 실습 체크리스트
- 2차원 배열로 간단한 게임 맵 데이터를 표현한다.

## 본문

### 16-1 다차원 배열 활용
- 2차원 배열은 행(row)과 열(column) 구조의 데이터를 표현할 때 유용합니다.
- 게임 맵, 좌석표, 성적표 같은 문제에 자주 사용됩니다.

### 16-2 3차원 배열
- 3차원 배열은 층(layer) 개념이 추가된 구조입니다.
- 좌표계(x, y, z)나 시간 축이 있는 데이터 표현에 활용할 수 있습니다.

예시 코드:
```c
#include <stdio.h>

int main(void) {
    int map[2][3] = {
        {1, 0, 1},
        {0, 1, 0}
    };

    for (int r = 0; r < 2; r++) {
        for (int c = 0; c < 3; c++) {
            printf("%d ", map[r][c]);
        }
        printf("\n");
    }
    return 0;
}
```
