## 본문

### 02-1 Hello World! 출력해보기
- C 프로그램은 보통 `main` 함수에서 시작합니다. `main`은 프로그램의 "출발점"입니다.
- 가장 먼저 `Hello, World!`를 출력해보는 이유는 개발 환경이 정상 동작하는지 확인하기 위해서입니다.
- 첫 실행이 성공하면 이후 문법 학습에서 "환경 문제"와 "코드 문제"를 구분하기 쉬워집니다.

```c
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```
- Visual Studio에서는 `Ctrl + Shift + B`로 빌드, `Ctrl + F5`로 실행하면 결과를 바로 확인할 수 있습니다.

### 02-2 주석이란?
- 주석은 컴퓨터가 실행하지 않고, 사람이 읽기 위한 메모입니다.
- 한 줄 주석은 `//`, 여러 줄 주석은 `/* ... */`를 사용합니다.
- 초보자는 "왜 이렇게 작성했는지"를 짧게 남기는 습관을 들이면 복습 속도가 빨라집니다.

```c
#include <stdio.h>

int main(void) {
    // 한 줄 주석: 아래 문장은 화면에 인사말을 출력합니다.
    printf("안녕하세요!\n");

    /*
      여러 줄 주석:
      프로그램의 목적, 작성자, 수정 날짜처럼
      길게 설명할 내용을 정리할 때 사용합니다.
    */
    return 0;
}

//main 밖에도 자유롭게 주석을 작성할 수 있습니다.
```

### 02-3 `printf`로 다양한 텍스트 출력
- `printf`는 화면(콘솔)에 글자를 출력하는 가장 기본 함수입니다.
- `\n`은 줄바꿈을 의미하며, 출력 결과를 보기 좋게 정리할 때 자주 사용합니다.
- 이 챕터에서는 문자열 출력과 줄바꿈까지만 익히고, 형식 지정자(`%d`, `%f` 등)는 변수 챕터에서 학습합니다.

```c
#include <stdio.h>

int main(void) {
    printf("안녕하세요.\n");
    printf("C 언어를 시작합니다.\n");
    printf("줄바꿈 연습 중입니다.\n");

    return 0;
}
```

예상 출력:
```text
안녕하세요.
C 언어를 시작합니다.
줄바꿈 연습 중입니다.
```

연습문제:
1. `Hello, World!`를 출력하는 코드를 직접 작성해보세요.
2. 한 줄/여러 줄 주석을 각각 1번 이상 사용해보세요.
3. `printf`로 본인 소개를 3줄 이상 출력해보세요. (줄바꿈 포함)

정답 포인트:
- 프로그램 시작 함수는 `main`입니다.
- 주석 문법: `//`, `/* ... */`
- 줄바꿈은 `\n`을 사용합니다.
