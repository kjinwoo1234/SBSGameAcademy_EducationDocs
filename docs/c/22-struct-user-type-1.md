### 22-2 구조체/배열/포인터

## 실습 체크리스트
- 학생 정보 구조체 배열을 만들어 검색 기능을 구현한다.

## 본문

### 22-1 구조체 기초
- 구조체는 서로 다른 타입 데이터를 하나로 묶는 사용자 정의 자료형입니다.
- 멤버 접근은 점(`.`) 연산자, 포인터는 화살표(`->`) 연산자를 사용합니다.

### 22-2 구조체/배열/포인터
- 구조체 배열은 여러 개의 객체를 묶어 관리할 때 효과적입니다.
- 구조체 포인터로 데이터를 전달하면 복사 비용을 줄일 수 있습니다.

멤버 접근 방식:
| 상황 | 접근 연산자 | 예시 |
|---|---|---|
| 구조체 변수 | `.` | `s.id` |
| 구조체 포인터 | `->` | `p->id` |

예시 코드:
```c
#include <stdio.h>

struct Student {
    int id;
    char grade;
    int score;
};

int main(void) {
    struct Student s[2] = {{1, 'A', 95}, {2, 'B', 82}};
    for (int i = 0; i < 2; i++) {
        printf("%d %c %d\n", s[i].id, s[i].grade, s[i].score);
    }
    return 0;
}
```

예상 출력:
```text
1 A 95
2 B 82
```
