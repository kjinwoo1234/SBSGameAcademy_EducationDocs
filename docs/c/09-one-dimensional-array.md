# Chapter 09 1차원 배열

## 학습 목표
- 1차원 배열을 선언·초기화하고 인덱스로 칸에 접근한다.
- 배열 칸 개수를 `sizeof`로 확인하는 방법을 익힌다.
- 문자열과 문자 배열의 관계를 이해한다.

## 본문

### 09-1 배열 선언

변수는 값 하나를 담는 상자에 가깝습니다. 그런데 점수처럼 같은 종류의 값이 여러 개면 `score1`, `score2`처럼 변수를 계속 늘리기 어렵습니다. 이럴 때 쓰는 것이 **배열**입니다. **배열**은 같은 자료형의 값을 메모리에 **연속된 칸**으로 모아 두고, 이름 하나로 그 칸들을 다루는 자료 구조입니다.

먼저 칸 개수만 정해 만드는 방법입니다. 아래 한 줄의 의미를 하나씩 짚어 보겠습니다.

```c
int scores[5];
```

- `int` — 각 칸에 들어갈 자료형입니다.
- `scores` — 배열의 이름입니다.
- `[5]` — 칸이 **5개**라는 뜻입니다.

이 줄까지 쓰면 칸 다섯 개가 생깁니다. 다만 `main` 안의 지역 배열은 아직 값을 넣지 않으면 내용이 정해져 있지 않습니다. 원하는 값은 `scores[0] = 80;`처럼 칸마다 넣습니다.

아래를 실행해 보세요.

```c
#include <stdio.h>

int main(void) {
    int scores[5];
    scores[0] = 80;
    scores[1] = 95;
    printf("%d\n", scores[0]);
    return 0;
}
```

**예상 출력**

```text
80
```

**코드 해석**
- `int scores[5];`로 칸 5개를 만든 뒤 `scores[0]`에 `80`을 넣습니다.
- `scores[0]`을 출력하면 `80`이 나옵니다.

이어서 처음부터 값을 넣어 만드는 방법입니다. **초기화 목록**을 씁니다. 아래 한 줄의 의미를 하나씩 짚어 보겠습니다.

```c
int scores[4] = { 60, 85, 70, 90 };
```

- `int scores[4]` — 정수 칸 4개짜리 배열 `scores`를 선언합니다.
- `{ 60, 85, 70, 90 }` — 처음부터 넣을 값을 중괄호 안에 나열합니다.

값이 네 개이고 `[4]`도 네 칸이므로 크기가 맞습니다. 크기를 생략해 `int scores[] = { 60, 85, 70, 90 };`처럼 쓰면, 적어 둔 값의 개수가 곧 배열 크기가 됩니다.

아래를 실행해 보세요.

```c
#include <stdio.h>

int main(void) {
    int scores[] = { 60, 85, 70, 90 };
    printf("%d\n", scores[0]);
    return 0;
}
```

**예상 출력**

```text
60
```

**코드 해석**
- 첫 칸 `scores[0]`은 `60`입니다.
- 값 네 개로 초기화했으므로 칸도 네 개입니다.

### 09-2 칸 개수 확인

배열이 몇 칸인지는 직접 세거나, **`sizeof`**로 계산할 수 있습니다. `sizeof(scores)`는 배열 전체 바이트 크기이고, `sizeof(scores[0])`는 칸 하나 바이트 크기입니다. 둘을 나누면 칸 개수가 됩니다.

아래를 실행해, 크기를 명시한 배열과 초기화 목록으로 만든 배열의 칸 개수를 비교해 보세요.

```c
#include <stdio.h>

int main(void) {
    int a[5];
    int b[] = { 60, 85, 70, 90 };

    printf("%zu\n", sizeof(a) / sizeof(a[0]));
    printf("%zu\n", sizeof(b) / sizeof(b[0]));
    return 0;
}
```

**예상 출력**

```text
5
4
```

**코드 해석**
- `a`는 `[5]`이므로 칸 개수는 `5`입니다.
- `b`는 값이 네 개이므로 칸 개수는 `4`입니다.

### 09-3 인덱스 접근

**인덱스**는 각 칸의 위치 번호이며 **0부터** 시작합니다. 네 칸짜리 배열이면 쓸 수 있는 번호는 `0`, `1`, `2`, `3`입니다. `scores[0]`은 첫 칸이고, 마지막 칸은 `scores[칸개수 - 1]`입니다.

아래를 실행해 첫 값과 마지막 값을 확인해 보세요.

```c
#include <stdio.h>

int main(void) {
    int scores[] = { 80, 95, 70, 100 };
    int n = (int)(sizeof(scores) / sizeof(scores[0]));

    printf("%d\n", scores[0]);
    printf("%d\n", scores[n - 1]);
    return 0;
}
```

**예상 출력**

```text
80
100
```

**코드 해석**
- `scores[0]`은 첫 값 `80`입니다.
- `n`은 `4`이므로 `scores[n - 1]`은 `scores[3]`이고, 마지막 값 `100`입니다.

존재하지 않는 번호로 접근하면 안 됩니다. 네 칸짜리에서 `scores[4]`는 범위 밖입니다. C는 이런 접근을 컴파일 단계에서 항상 막아 주지 않아, **런타임**에 잘못된 동작으로 이어질 수 있습니다.

### 09-4 순회

배열의 모든 칸을 살펴보는 것을 **순회**라고 합니다. 앞에서 배운 `for`로 인덱스를 `0`부터 칸 개수 직전까지 돌리면 됩니다.

아래를 실행해 보세요.

```c
#include <stdio.h>

int main(void) {
    int scores[] = { 80, 95, 70, 100 };
    int n = (int)(sizeof(scores) / sizeof(scores[0]));
    int i;

    for (i = 0; i < n; i++) {
        printf("%d: %d\n", i, scores[i]);
    }
    return 0;
}
```

**예상 출력**

```text
0: 80
1: 95
2: 70
3: 100
```

**코드 해석**
- `i`는 `0`부터 `n` 직전까지입니다. 네 칸이면 `0`~`3`입니다.
- `scores[i]`로 인덱스와 값을 함께 다룹니다.

합을 구할 때도 같은 방식으로 순회합니다.

```c
#include <stdio.h>

int main(void) {
    int nums[] = { 10, 20, 30, 40, 50 };
    int n = (int)(sizeof(nums) / sizeof(nums[0]));
    int i, sum = 0;

    for (i = 0; i < n; i++) {
        sum += nums[i];
    }
    printf("합: %d\n", sum);
    return 0;
}
```

**예상 출력**

```text
합: 150
```

**코드 해석**
- 다섯 칸을 돌며 `sum`에 더합니다.
- 합은 `150`입니다.

### 09-5 배열 기반 문자열

C에서 흔히 말하는 문자열은 `char` 배열이며, 끝에는 **널 문자** `\0`이 들어갑니다. “문자열이 차지하는 글자 수”와 “배열을 몇 칸 잡았는지”는 다른 개념이므로, 배열 크기는 보통 문자 개수보다 **최소 1칸 더** 잡아 `\0`을 둘 자리를 남깁니다.

`\0`은 “여기까지가 문자열”이라고 표시하는 특수 문자입니다. `printf("%s", ...)`나 `strlen` 같은 함수는 `\0`을 만날 때까지를 한 덩어리의 문자열로 봅니다. 예를 들어 `char name[4] = "Kim";`은 `K`, `i`, `m`, `\0` 네 칸을 쓰는 그림으로 이해할 수 있습니다.

| 항목 | 정수 배열 | 문자열 배열 |
|---|---|---|
| 타입 | `int` 배열 | `char` 배열 |
| 끝 표시 | 없음 | `\0` 있음 |
| 출력 방식 | 반복문 순회 | `%s` 가능 |

아래를 실행해 보세요.

```c
#include <stdio.h>

int main(void) {
    char name[10] = "Kim";
    printf("이름: %s\n", name);
    return 0;
}
```

**예상 출력**

```text
이름: Kim
```

**코드 해석**
- `name`은 `char` 배열에 문자열을 담은 것입니다.
- `%s`는 `\0`까지를 문자열로 출력합니다.

### 연습문제

**문제 1 — 배열 최댓값**
- 문제: 정수 다섯 개를 배열에 저장한 뒤 최댓값을 출력하세요.
- 입력: 정수 5개
- 출력: 최댓값 1개
- 조건(힌트): 반복문으로 배열 전체를 순회하며 현재 최댓값을 갱신하세요.

**문제 2 — 배열 평균**
- 문제: 정수 다섯 개의 평균을 소수 둘째 자리까지 출력하세요.
- 입력: 정수 5개
- 출력: 평균값(소수 둘째 자리까지)
- 조건(힌트): `(double)sum / 5`처럼 실수 나눗셈이 되도록 캐스팅하세요.

**문제 3 — 문자열 길이**
- 문제: 공백 없는 문자열 하나를 입력받아 길이를 출력하세요.
- 입력: 문자열 1개
- 출력: 문자열 길이 1개
- 조건(힌트): `strlen`을 사용하거나 직접 반복문으로 길이를 계산하세요.

### 정답 포인트

반복문 인덱스는 `i < 배열크기`처럼 범위를 엄격히 맞추고, 평균은 `(double)합 / 개수`처럼 실수 나눗셈이 되도록 조정합니다.
