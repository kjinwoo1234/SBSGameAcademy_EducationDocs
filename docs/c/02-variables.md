# C 변수와 자료형

변수는 데이터를 저장하는 메모리 공간입니다.  
자료형(type)은 "어떤 값인지"와 "얼마나 큰 메모리를 쓸지"를 결정합니다.

## 학습 목표
- C 기본 자료형의 용도와 차이를 설명할 수 있다.
- 선언/초기화/재할당 흐름을 코드로 작성할 수 있다.
- 형 변환과 오버플로우 관련 실수를 점검할 수 있다.

## 핵심
- 기본 자료형: `int`, `float`, `double`, `char`
- 형 변환: `(double)a / b`처럼 명시적 캐스팅
- 안정성: 초기화와 범위 점검 습관

## 1) 기본 자료형
- `int`: 정수
- `float`: 단정도 실수
- `double`: 배정도 실수 (정밀도 높음)
- `char`: 문자 1개

```c
int age = 20;
float rate = 3.14f;      // float 리터럴은 f 권장
double pi = 3.141592;
char grade = 'A';
```

## 2) 선언, 초기화, 재할당
```c
int score;       // 선언
score = 90;      // 대입
int level = 1;   // 선언 + 초기화
level = 2;       // 재할당
```

## 3) 형 변환 (Type Casting)
정수끼리 나눗셈하면 결과도 정수입니다.  
실수 결과가 필요하면 명시적 형 변환을 사용합니다.

```c
int a = 7, b = 2;
double wrong = a / b;          // 3.0
double correct = (double)a / b; // 3.5
```

## 4) 오버플로우와 범위
자료형마다 표현 가능한 범위가 있습니다.  
범위를 넘어가면 오버플로우가 발생할 수 있습니다.

```c
int max = 2147483647;
max = max + 1;   // 오버플로우 가능
```

## 5) 자주 하는 실수
- `float`에 `3.14`를 넣고 `f` 접미사를 빼먹음
- 정수 나눗셈 결과를 실수라고 오해함
- 초기화하지 않은 변수를 사용함

## 단계별 실습
1. `int`, `float`, `double`, `char` 변수를 각각 선언/초기화한다.
2. 정수 나눗셈과 실수 나눗셈 결과를 비교 출력한다.
3. 오버플로우 가능 상황을 코드로 확인하고 주석으로 원인을 정리한다.

## 6) 연습 문제
1. 반지름을 입력받아 원의 넓이를 `double`로 계산하세요.
2. 두 정수의 몫/나머지/실수 나눗셈 결과를 모두 출력하세요.
3. `char` 2개를 입력받아 각각 출력하세요.

## 셀프 퀴즈

## 정답

2. 정수끼리 먼저 연산되어 3이 계산된 뒤 `double`로 변환되기 때문
