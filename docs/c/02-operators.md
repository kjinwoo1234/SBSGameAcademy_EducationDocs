# [C 자습 자료] 02. 연산자

## 1. 산술 연산자
`+`, `-`, `*`, `/`, `%`

```c
int x = 10, y = 3;
printf("%d\n", x + y); // 13
printf("%d\n", x % y); // 1
```

---

## 2. 비교/논리 연산자
- 비교: `==`, `!=`, `>`, `<`, `>=`, `<=`
- 논리: `&&`, `||`, `!`

```c
int score = 85;
int pass = (score >= 60) && (score <= 100);
```

---

## 3. 증감 연산자
`++`, `--`의 전위/후위 동작 차이를 이해합니다.

```c
int n = 5;
int a = ++n; // n=6, a=6
int b = n++; // b=6, n=7
```
