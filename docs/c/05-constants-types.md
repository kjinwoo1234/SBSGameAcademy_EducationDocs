## 본문

### 05-1 기본 자료형
- 자료형은 "어떤 종류의 데이터를 저장할지"를 정하는 규칙입니다.
- `char`는 문자, `int`는 정수, `float`/`double`은 실수를 저장합니다.
- 자료형마다 저장 가능한 범위와 정밀도가 다르므로 상황에 맞게 선택해야 합니다.
- 자료형 크기는 컴파일러/플랫폼에 따라 달라질 수 있으므로, 실제 프로젝트에서는 `sizeof`로 확인하는 습관이 좋습니다.

입문 4대 타입(우선 학습):
- `char`: 문자 1개
- `int`: 정수
- `float`: 실수(단정밀도)
- `double`: 실수(배정밀도)

확장 타입:
| 타입 | 용도 | 비고 |
|---|---|---|
| `short` | 작은 범위 정수 | 메모리를 조금 아끼고 싶을 때 |
| `long`, `long long` | 큰 범위 정수 | 큰 숫자 계산/카운트 |
| `unsigned int` | 음수 없는 정수 | 범위를 양수 쪽으로 확장 |
| `signed/unsigned char` | 문자/바이트 데이터 | 바이너리 처리에서 자주 사용 |
| `long double` | 더 높은 실수 정밀도 | 환경 의존성 큼 |
| `_Bool` (`bool`) | 참/거짓 저장 | `stdbool.h` 포함 시 `bool` 사용 가능 |
| `void` | 값 없음 | 반환값 없음, 범용 포인터(`void*`) |

크기 확인 예시(일반적인 Windows MSVC 기준):
```c
#include <stdio.h>

int main(void) {
    printf("char: %zu byte\n", sizeof(char));
    printf("int: %zu byte\n", sizeof(int));
    printf("float: %zu byte\n", sizeof(float));
    printf("double: %zu byte\n", sizeof(double));
    return 0;
}
```

### 05-2 문자 표현 방식
- 컴퓨터는 문자를 숫자 코드(예: ASCII)로 저장합니다.
- `'A'` 같은 문자 리터럴도 내부적으로는 정수 값으로 처리됩니다.
- 문자와 숫자 관계를 이해하면 입력/출력, 문자열 처리에서 실수가 줄어듭니다.

### 05-3 상수
- 상수는 실행 중 값이 바뀌지 않는 데이터입니다.
- 예를 들어 `100`, `3.14`, `'K'`, `"Hello"`는 각각 정수/실수/문자/문자열 상수입니다.
- 의미 있는 고정값은 상수로 관리하면 코드 가독성과 안정성이 좋아집니다.
- `const`로 선언한 값은 재할당할 수 없으므로, 변경되면 안 되는 설정값에 사용합니다.

### 05-4 자료형 변환
- 자료형 변환은 한 타입의 값을 다른 타입으로 바꾸는 과정입니다.
- 자동 형 변환은 컴파일러가 알아서 처리하고, 명시적 형 변환(캐스팅)은 개발자가 의도를 직접 표현합니다.
- 정수 나눗셈처럼 타입에 따라 결과가 달라질 수 있으므로, 계산 전에 변환이 필요한지 확인해야 합니다.
- 실수 비교에서는 부동소수점 오차가 있을 수 있어 `==` 비교를 바로 쓰지 않도록 주의합니다.

예시 코드:
```c
#include <stdio.h>

int main(void) {
    int a = 7, b = 2;
    double result1 = a / b;           // 정수 나눗셈 후 double 저장
    double result2 = (double)a / b;   // 실수 나눗셈

    const int MAX_LIFE = 3;
    char grade = 'A';

    printf("result1: %.2f\n", result1); // 3.00
    printf("result2: %.2f\n", result2); // 3.50
    printf("grade ASCII: %d\n", grade); // 65
    printf("MAX_LIFE: %d\n", MAX_LIFE);
    return 0;
}
```

연습문제:
1. `int x = 5, y = 2;`일 때 `x / y`와 `(double)x / y`의 차이를 출력해보세요.
2. 문자 `'Z'`의 ASCII 코드를 출력해보세요.
3. `const`를 사용해 `PI = 3.14`를 선언해보세요.

정답 예시:
```c
double a = x / y;
double b = (double)x / y;
printf("%.2f %.2f\n", a, b);
printf("%d\n", 'Z');
const double PI = 3.14;
```

자주 하는 실수:
- `const` 변수에 다시 값을 넣으려는 시도
- 정수 나눗셈 결과를 실수로 기대하는 실수 (`5 / 2`는 `2`)
- `%d`, `%f` 형식 지정자를 변수 타입과 다르게 쓰는 실수
