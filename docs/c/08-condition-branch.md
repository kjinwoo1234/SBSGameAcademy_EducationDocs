## 본문

### 08-1 조건적 실행과 분기

조건문은 상황에 따라 **서로 다른 코드**를 실행하게 만드는 도구입니다. `if`, `else if`, `else`를 이어 붙이면 여러 단계의 조건을 위에서 아래로 검사할 수 있습니다. 조건식을 단순하고 읽기 쉽게 쓸수록 나중에 버그를 찾기도 쉬워집니다.

### 08-2 `continue` / `break`

`continue`는 **지금 반복**에서 남은 본문을 건너뛰고 곧바로 다음 반복으로 넘어갑니다. `break`는 반복문이나 `switch` 전체를 **즉시 종료**합니다. 두 키워드는 흐름을 세밀하게 조절할 때 유용하지만, 너무 많이 쓰면 읽는 사람이 따라가기 어려워질 수 있으므로 필요한 곳에만 쓰는 편이 좋습니다.

### 08-3 `switch`와 `goto`

`switch`는 **하나의 값**을 기준으로 여러 경우를 나눌 때 읽기 좋은 편입니다. 각 `case` 끝에서 `break`를 빼먹으면 다음 `case`까지 이어 실행되는 **fall-through**가 생길 수 있으므로, 의도가 아니라면 `break`를 넣는 습관이 안전합니다.

`goto`는 코드 실행을 **레이블(label)**이 붙은 위치로 즉시 옮깁니다. 레이블은 `이름:` 형태로 붙이고, `goto 이름;`으로 점프합니다. 매우 제한적인 탈출이나 예외적인 구조에서 가끔 쓰이지만, 남용하면 흐름이 꼬여 유지보수가 어려워지므로 학습 단계에서는 “있다” 정도로만 이해해도 됩니다.

```c
#include <stdio.h>

int main(void) {
    int n = 0;

start:  // 레이블
    n++;
    printf("n = %d\n", n);
    if (n < 3) goto start;  // start 레이블 위치로 이동

    return 0;
}
```

분기 방식을 비교하면 다음과 같습니다.

| 방식 | 적합한 상황 | 장점 | 주의점 |
|---|---|---|---|
| `if / else if / else` | 범위 조건, 복합 조건 | 유연함 | 조건식이 길어지면 가독성 저하 |
| `switch` | 값 1개에 대한 다중 분기 | 읽기 쉬움 | `break` 누락 시 fall-through |
| `goto` | 매우 제한적인 탈출 흐름 | 즉시 이동 가능 | 남용 시 유지보수 어려움 |

**fall-through**는 `switch`에서 `break`를 쓰지 않아 다음 `case` 코드까지 연달아 실행되는 현상입니다. 의도한 경우가 아니면 버그 원인이 되기 쉽습니다.

```c
#include <stdio.h>

int main(void) {
    int score = 85;

    if (score >= 90) printf("A\n");
    else if (score >= 80) printf("B\n");
    else printf("C 이하\n");

    int menu = 2;
    switch (menu) {
        case 1: printf("시작\n"); break;
        case 2: printf("설정\n"); break;
        default: printf("종료\n"); break;
    }
    return 0;
}
```

`if`는 점수 구간을 위에서부터 나누어 검사하고, `switch`는 `menu` 값 하나로 분기합니다. 각 `case` 뒤의 `break`는 그 분기만 끝내고 `switch` 밖으로 나가게 합니다.

**예상 출력**

```text
B
설정
```

### 연습문제

**문제 1**  
점수(`0~100`)를 입력받아 A/B/C/F를 출력하세요.

**문제 2**  
숫자 1~3을 입력받아 각 메뉴 이름을 `switch`로 출력하세요.

**문제 3**  
반복문에서 3의 배수는 건너뛰고(`continue`), 20 이상이면 종료(`break`)하는 코드를 작성하세요.

### 정답 포인트

등급 분기는 `if (score >= 90) ... else if ...` 형태로, 메뉴 분기는 `switch (menu) { case ... }`로 쓰면 됩니다. `continue`와 `break`는 반복문 안에서 의도한 위치에 두어야 합니다.
