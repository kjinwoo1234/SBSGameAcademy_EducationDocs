- 문자열 입력 후 길이/비교를 수행하는 미니 도구를 작성한다.

## 본문

### 21-1 스트림과 데이터 이동
- 표준 입력/출력/에러 스트림(`stdin`, `stdout`, `stderr`)의 역할을 먼저 이해합니다.
- 대부분의 콘솔 입출력은 이 스트림을 통해 처리됩니다.

### 21-2 문자 단위 입출력
- `getchar`, `putchar`는 문자 1개 단위 입출력에 사용됩니다.
- 작은 단위 제어가 필요할 때 유용합니다.

### 21-3 문자열 단위 입출력
- `fgets`는 공백 포함 문자열을 안전하게 입력받기 좋습니다.
- `puts`는 문자열 출력 후 자동 줄바꿈을 해줍니다.

### 21-4 표준 입출력과 버퍼
- 입력 버퍼에 남은 개행 문자(`\n`) 때문에 다음 입력이 건너뛰는 문제가 자주 발생합니다.
- 함수 조합 순서와 버퍼 처리 습관이 중요합니다.

### 21-5 문자열 관련 함수
- `strlen`, `strcpy`, `strcat`, `strcmp`는 가장 자주 쓰는 문자열 함수입니다.
- 버퍼 크기 확인 없이 복사/연결하면 오버플로우 위험이 있습니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 버퍼(buffer) | 입력/출력을 잠시 저장해두는 메모리 공간 |
| 오버플로우(overflow) | 저장 공간을 초과해 데이터가 넘치는 오류 |

문자열 함수 요약:
| 함수 | 용도 |
|---|---|
| `strlen` | 문자열 길이 계산 |
| `strcmp` | 문자열 비교 |
| `strcpy` | 문자열 복사 |
| `strcat` | 문자열 이어붙이기 |

예시 코드:
```c
#include <stdio.h>
#include <string.h>

int main(void) {
    char a[30], b[30];
    fgets(a, sizeof(a), stdin);
    fgets(b, sizeof(b), stdin);

    a[strcspn(a, "\n")] = '\0';
    b[strcspn(b, "\n")] = '\0';

    printf("len(a): %zu\n", strlen(a));
    printf("compare: %d\n", strcmp(a, b));
    return 0;
}
```

예상 출력(입력: `abc`, `abd`):
```text
len(a): 3
compare: -1
```

연습문제:
1. 문제 1 - 문자열 길이 출력
   - 문제: 두 문자열을 입력받아 각각 길이를 출력하세요.
   - 입력: 문자열 2개
   - 출력: `len1`, `len2`
   - 조건: 개행 문자 제거 후 길이 계산
2. 문제 2 - 문자열 비교
   - 문제: 두 문자열이 같으면 `same`, 다르면 `different`를 출력하세요.
   - 입력: 문자열 2개
   - 출력: 비교 결과 문자열
   - 조건: `==` 대신 `strcmp` 사용

정답 포인트:
- `fgets` 사용 시 개행 제거
- 비교는 `==`가 아니라 `strcmp`
