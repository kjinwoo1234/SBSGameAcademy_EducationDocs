### 21-1 스트림과 데이터 이동

### 21-3 문자열 단위 입출력

### 21-4 표준 입출력과 버퍼

## 실습 체크리스트
- 문자열 입력 후 길이/비교를 수행하는 미니 도구를 작성한다.

## 본문

### 21-1 스트림과 데이터 이동
- 표준 입력/출력/에러 스트림(`stdin`, `stdout`, `stderr`)의 역할을 먼저 이해합니다.
- 대부분의 콘솔 입출력은 이 스트림을 통해 처리됩니다.

### 21-2 문자 단위 입출력
- `getchar`, `putchar`는 문자 1개 단위 입출력에 사용됩니다.
- 작은 단위 제어가 필요할 때 유용합니다.

### 21-3 문자열 단위 입출력
- `fgets`는 공백 포함 문자열을 안전하게 입력받기 좋습니다.
- `puts`는 문자열 출력 후 자동 줄바꿈을 해줍니다.

### 21-4 표준 입출력과 버퍼
- 입력 버퍼에 남은 개행 문자(`\n`) 때문에 다음 입력이 건너뛰는 문제가 자주 발생합니다.
- 함수 조합 순서와 버퍼 처리 습관이 중요합니다.

### 21-5 문자열 관련 함수
- `strlen`, `strcpy`, `strcat`, `strcmp`는 가장 자주 쓰는 문자열 함수입니다.
- 버퍼 크기 확인 없이 복사/연결하면 오버플로우 위험이 있습니다.

예시 코드:
```c
#include <stdio.h>
#include <string.h>

int main(void) {
    char a[30], b[30];
    fgets(a, sizeof(a), stdin);
    fgets(b, sizeof(b), stdin);

    a[strcspn(a, "\n")] = '\0';
    b[strcspn(b, "\n")] = '\0';

    printf("len(a): %zu\n", strlen(a));
    printf("compare: %d\n", strcmp(a, b));
    return 0;
}
```

연습문제:
1. 두 문자열을 입력받아 길이를 각각 출력
2. 두 문자열이 같은지 비교 출력

정답 포인트:
- `fgets` 사용 시 개행 제거
- 비교는 `==`가 아니라 `strcmp`
