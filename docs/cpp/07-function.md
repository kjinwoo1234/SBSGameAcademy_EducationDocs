# C++ 함수

함수는 프로그램을 작고 명확한 단위로 분리합니다.  
재사용성, 테스트 용이성, 유지보수성이 크게 향상됩니다.

## 1) 기본 함수
```cpp
int Add(int a, int b) {
    return a + b;
}
```

## 2) 참조 전달
원본 수정이 필요하거나 복사 비용을 줄일 때 참조를 사용합니다.

```cpp
void Increase(int& x) {
    x++;
}
```

## 3) 상수 참조
읽기 전용으로 전달하면서 복사 비용을 줄입니다.

```cpp
void PrintName(const std::string& name) {
    std::cout << name << '\n';
}
```
