# Chapter 11 파일 입출력

## 학습 목표
- 파일 스트림을 사용해 텍스트 파일을 읽고 쓸 수 있다.
- 파일 열기 실패 처리와 자원 정리 습관을 익힌다.

## 세부 주제
### 11-1 파일 스트림 기초
- `ofstream`, `ifstream`

### 11-2 파일 쓰기/읽기
- 열기, 기록, 닫기

### 11-3 예외/오류 처리
- 열기 실패 체크

## 실습 체크리스트
- 사용자 정보를 파일에 저장하고 다시 읽어 출력한다.

## 본문

### 11-1 파일 스트림 기초
- `ofstream`은 파일 출력(쓰기), `ifstream`은 파일 입력(읽기)에 사용합니다.
- `<fstream>` 헤더를 포함하면 파일 스트림 클래스를 사용할 수 있습니다.

### 11-2 파일 쓰기/읽기
- 쓰기 흐름은 `열기 -> 기록 -> 닫기` 순서입니다.
- 읽기 시에는 `getline`으로 한 줄씩 가져와 처리할 수 있습니다.

### 11-3 예외/오류 처리
- 파일 열기에 실패하면 경로/권한/파일 존재 여부를 먼저 확인해야 합니다.
- 스트림 상태를 `if (!fin)`처럼 검사해 안전하게 분기합니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 파일 스트림(file stream) | 파일과 프로그램 사이 데이터 이동 통로 |
| 상대경로 | 현재 실행 위치를 기준으로 해석되는 파일 경로 |

파일 입출력 흐름:
| 단계 | 핵심 작업 | 체크 포인트 |
|---|---|---|
| 열기 | `ofstream`/`ifstream` 생성 | `if (!stream)` 실패 확인 |
| 처리 | 읽기/쓰기 수행 | 포맷 일치 여부 확인 |
| 닫기 | `close()` 또는 범위 종료 | 데이터 flush 여부 확인 |

예시 코드:
```cpp
#include <fstream>
#include <iostream>
#include <string>
using namespace std;

int main() {
    ofstream fout("player.txt");
    if (!fout) return 1;
    fout << "Kim 120\n";
    fout.close();

    ifstream fin("player.txt");
    if (!fin) return 1;
    string name;
    int hp = 0;
    fin >> name >> hp;
    cout << name << ' ' << hp << endl;
    return 0;
}
```

예상 출력:
```text
Kim 120
```

코드 해석:
- 먼저 `ofstream`으로 `player.txt`를 열고 이름/체력을 저장합니다.
- 다음에 `ifstream`으로 같은 파일을 다시 열어 값을 읽습니다.
- 스트림 열기 실패 시 즉시 `return 1;`로 종료해 오류 확산을 막습니다.
