# Chapter 01 출력과 기본 입출력

## 학습 목표
- C++의 기본 출력/입력 흐름을 이해한다.
- `cout`, `cin`, `endl`의 역할을 구분할 수 있다.

## 세부 주제
### 01-1 C++ 프로그램의 시작점
- `main` 함수와 실행 흐름

### 01-2 `cout`으로 출력하기
- 스트림 출력과 줄바꿈

### 01-3 `cin`으로 입력받기
- 변수와 입력 연결

## 실습 체크리스트
- 이름과 나이를 입력받아 문장으로 출력해본다.

## 본문

### 01-1 C++ 프로그램의 시작점
- C++ 프로그램은 보통 `main` 함수에서 시작합니다.
- `#include <iostream>`은 표준 입출력 기능을 사용하기 위한 헤더 포함입니다.

### 01-2 `cout`으로 출력하기
- `cout`은 콘솔(화면)로 값을 보내는 출력 스트림입니다.
- `<<` 연산자는 값을 출력 스트림으로 흘려보낼 때 사용합니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 스트림(stream) | 데이터가 이동하는 통로 |
| `endl` | 줄바꿈 후 출력 버퍼를 비우는 조작자 |

### 01-3 `cin`으로 입력받기
- `cin`은 키보드 입력을 읽어 변수에 저장하는 입력 스트림입니다.
- `>>` 연산자는 입력 스트림에서 값을 꺼내 변수에 넣을 때 사용합니다.

입력 방식 비교:
| 방식 | 특징 | 적합한 상황 |
|---|---|---|
| `cin >> 변수` | 공백 전까지만 입력 | 숫자/단일 단어 입력 |
| `getline(cin, 문자열)` | 공백 포함 한 줄 전체 입력 | 문장 입력 |

예시 코드:
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    int age = 0;

    cout << "이름 입력: ";
    cin >> name;
    cout << "나이 입력: ";
    cin >> age;

    cout << "안녕하세요, " << name << "님. " << age << "살이군요." << endl;
    return 0;
}
```

예상 출력:
```text
이름 입력: Kim
나이 입력: 20
안녕하세요, Kim님. 20살이군요.
```

코드 해석:
- `string name; int age = 0;`에서 입력값을 담을 변수를 준비합니다.
- `cin >> name`, `cin >> age`로 입력 스트림에서 값을 꺼내 변수에 저장합니다.
- 마지막 `cout` 체인으로 문자열과 숫자를 한 문장으로 결합해 출력합니다.

연습문제:
1. 문제 1 - 두 수 출력

   - 출력: 합계 1개
   - 조건: `cout`으로 안내 문장 포함
2. 문제 2 - 자기소개 출력
   - 문제: 이름, 전공을 입력받아 자기소개 한 줄을 출력하세요.
   - 입력: 이름, 전공
   - 출력: 자기소개 문장
   - 힌트: 출력 연산자 `<<`를 여러 번 연결

정답 포인트:

- 문자열과 숫자를 함께 출력할 때 `<<` 체인 사용
