# Chapter 04 조건문과 반복문

## 학습 목표
- `if`, `switch`, `for`, `while`의 사용 기준을 이해한다.
- 분기와 반복을 조합해 간단한 로직을 구현할 수 있다.

## 세부 주제
### 04-1 조건문
- `if/else`, `switch`

### 04-2 반복문
- `for`, `while`

### 04-3 흐름 제어
- `break`, `continue`

## 실습 체크리스트
- 점수 입력 시 학점 판정 프로그램을 작성한다.

## 본문

### 04-1 조건문
- `if/else`는 범위 조건(예: 90점 이상)에 유리합니다.
- `switch`는 고정된 값 분기에 유리합니다.

### 04-2 반복문
- `for`는 반복 횟수가 명확할 때 사용하기 좋습니다.
- `while`은 종료 조건이 실행 중 바뀌는 상황에 적합합니다.

### 04-3 흐름 제어
- `break`는 가장 가까운 반복/분기 블록을 즉시 종료합니다.
- `continue`는 현재 반복을 건너뛰고 다음 반복으로 이동합니다.

반복문 선택 가이드:
| 문법 | 적합한 상황 | 핵심 포인트 |
|---|---|---|
| `for` | 반복 횟수가 명확함 | 시작/조건/증감을 한 줄에 표현 |
| `while` | 종료 시점이 실행 중 결정됨 | 조건 중심 제어 |

용어 설명:
| 용어 | 의미 |
|---|---|
| 분기(branch) | 조건에 따라 실행 경로가 갈라지는 구조 |
| 반복(loop) | 같은 코드를 조건이 참인 동안 되풀이 실행 |

예시 코드:
```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 0;
    cout << "점수 입력: ";
    cin >> score;

    if (score >= 90) cout << "A\n";
    else if (score >= 80) cout << "B\n";
    else cout << "C 이하\n";

    cout << "홀수만 출력: ";
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) continue;
        cout << i << ' ';
    }
    cout << endl;
    return 0;
}
```

예상 출력(입력: 85):
```text
점수 입력: 85
B
홀수만 출력: 1 3 5 7 9
```

코드 해석:
- `if / else if / else`로 점수 범위를 순서대로 검사해 학점을 출력합니다.
- `for`문에서 짝수는 `continue`로 건너뛰고 홀수만 출력합니다.
- 마지막 `endl`로 출력 버퍼를 정리하며 줄을 마칩니다.
