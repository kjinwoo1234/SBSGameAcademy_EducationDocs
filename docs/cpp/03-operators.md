# Chapter 03 연산자와 형 변환

## 학습 목표
- 산술/비교/논리 연산자의 동작을 이해한다.
- 형 변환의 필요성과 주의점을 익힌다.

## 세부 주제
### 03-1 기본 연산자
- 산술, 비교, 논리 연산

### 03-2 증감 연산과 우선순위
- 전위/후위 차이

### 03-3 형 변환
- 암시적/명시적 변환

## 실습 체크리스트
- 같은 식을 서로 다른 타입으로 계산해 결과 차이를 확인한다.

## 본문

### 03-1 기본 연산자
- 산술 연산자(`+`, `-`, `*`, `/`, `%`)는 숫자 계산에 사용합니다.
- 비교/논리 연산자는 조건식에서 참/거짓을 만듭니다.

### 03-2 증감 연산과 우선순위
- `++a`는 증가 후 사용, `a++`는 사용 후 증가입니다.
- 복잡한 식에서는 괄호를 써서 의도를 명확히 하는 것이 안전합니다.

### 03-3 형 변환
- 암시적 변환은 컴파일러가 자동으로 타입을 맞추는 과정입니다.
- 명시적 변환은 `(double)a` 또는 `static_cast<double>(a)`처럼 개발자가 의도적으로 변환합니다.

연산자 용도 비교:
| 분류 | 예시 | 주 용도 |
|---|---|---|
| 산술 연산자 | `+`, `-`, `*`, `/`, `%` | 수치 계산 |
| 비교 연산자 | `==`, `!=`, `>`, `<` | 조건식 생성 |
| 논리 연산자 | `&&`, `||`, `!` | 조건 결합/반전 |

용어 설명:
| 용어 | 의미 |
|---|---|
| 암시적 변환 | 컴파일러가 자동으로 수행하는 타입 변환 |
| 명시적 변환 | 개발자가 의도를 담아 직접 지정하는 타입 변환 |

예시 코드:
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 7, b = 2;
    cout << "int division: " << a / b << endl;
    cout << "double division: " << static_cast<double>(a) / b << endl;

    int x = 5;
    cout << "pre ++x: " << ++x << endl;
    cout << "post x++: " << x++ << endl;
    cout << "after x: " << x << endl;
    return 0;
}
```

예상 출력:
```text
int division: 3
double division: 3.5
pre ++x: 6
post x++: 6
after x: 7
```

코드 해석:
- `a / b`는 둘 다 `int`라 소수점이 버려져 `3`이 됩니다.
- `static_cast<double>(a) / b`는 실수 계산으로 바뀌어 `3.5`가 됩니다.
- `++x`와 `x++`는 증가 시점이 달라 출력 결과가 다르게 나타납니다.

   - 입력: 없음
   - 출력: 각 단계 값

정답 포인트:

- 형 변환은 계산 전에 적용해야 의도한 결과 확보 가능
