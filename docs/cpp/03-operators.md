# C++ 연산자

연산자는 계산, 비교, 조건 결합의 기초입니다.  
복잡한 식을 작성할수록 연산자 우선순위와 타입 변환 규칙이 중요해집니다.

## 1) 산술/증감 연산자
```cpp
int x = 10, y = 3;
std::cout << x + y << '\n';
std::cout << x % y << '\n';

int n = 5;
int a = ++n; // 전위
int b = n++; // 후위
```

## 2) 비교/논리 연산자
```cpp
int score = 85;
bool pass = (score >= 60) && (score <= 100);
std::cout << pass << '\n'; // 1(true)
```

## 3) 복합 대입 연산자
```cpp
int money = 1000;
money += 500;
money -= 200;
```

## 4) 연습 문제
1. 점수 3개 평균을 구해 합격 여부를 출력하세요.
2. 전위/후위 증가 결과를 각각 출력해 차이를 설명하세요.
