# Chapter 06 문자열 다루기

## 학습 목표
- C++ 문자열(`std::string`)의 기본 사용법을 익힌다.
- 문자열 입력/결합/길이 계산을 구현할 수 있다.

## 세부 주제
### 06-1 `std::string` 기초
- 선언, 대입, 길이

### 06-2 문자열 입력
- `cin`과 `getline` 차이

### 06-3 문자열 함수
- `size`, `substr`, `find`

## 실습 체크리스트
- 문장을 입력받아 단어 개수를 세는 프로그램을 작성한다.

## 본문

### 06-1 `std::string` 기초
- `std::string`은 가변 길이 문자열 객체입니다.
- C 스타일 문자열보다 결합/복사/길이 처리 코드가 단순해집니다.

### 06-2 문자열 입력
- `cin >> s`는 공백 전까지만 입력받습니다.
- `getline(cin, s)`는 공백을 포함해 한 줄 전체를 입력받습니다.

### 06-3 문자열 함수
- `size()`는 문자열 길이를 반환합니다.
- `substr(pos, len)`은 부분 문자열을 추출하고, `find`는 위치를 찾습니다.

문자열 입력 방식 비교:
| 방식 | 공백 처리 | 예시 |
|---|---|---|
| `cin >> s` | 공백 전까지만 읽음 | `"Kim Min"` 입력 시 `"Kim"`만 저장 |
| `getline(cin, s)` | 공백 포함 한 줄 전체 | `"Kim Min"` 전체 저장 |

용어 설명:
| 용어 | 의미 |
|---|---|
| 부분 문자열(substring) | 문자열의 일부분 |
| `string::npos` | `find` 실패를 나타내는 특수 값 |

예시 코드:
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string title;
    cout << "문장 입력: ";
    getline(cin, title);

    cout << "길이: " << title.size() << endl;
    cout << "앞 3글자: " << title.substr(0, 3) << endl;
    return 0;
}
```

예상 출력(입력: Cpp Basic):
```text
문장 입력: Cpp Basic
길이: 9
앞 3글자: Cpp
```

코드 해석:
- `getline(cin, title)`로 공백을 포함한 문장 전체를 입력받습니다.
- `title.size()`로 문자 개수를 계산해 길이를 출력합니다.
- `title.substr(0, 3)`은 인덱스 0부터 3글자를 잘라 보여줍니다.

연습문제:
1. 문제 1 - 문자열 결합
   - 문제: 이름과 성을 입력받아 전체 이름을 출력하세요.
   - 입력: 성, 이름
   - 출력: 결합된 전체 이름
   - 조건: `+` 연산자로 문자열 결합
2. 문제 2 - 단어 검색
   - 문제: 문장과 검색 단어를 입력받아 포함 여부를 출력하세요.
   - 입력: 문장 1개, 단어 1개
   - 출력: 포함/미포함 메시지
   - 힌트: `find` 반환값 비교

정답 포인트:
- 공백 포함 입력이 필요하면 `getline` 사용
- `find`가 실패하면 `string::npos` 반환
