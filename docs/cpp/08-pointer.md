# C++ 포인터

포인터는 메모리 주소를 직접 다루는 도구입니다.  
C++에서는 포인터를 이해하되, 소유권이 복잡한 경우 스마트 포인터도 함께 고려해야 합니다.

## 1) 기본 사용
```cpp
int n = 10;
int* p = &n;
std::cout << *p << '\n';
```

## 2) 포인터로 값 변경
```cpp
*p = 30;
std::cout << n << '\n'; // 30
```

## 3) 배열과 포인터
```cpp
int arr[3] = {10, 20, 30};
int* q = arr;
std::cout << *(q + 1) << '\n'; // 20
```

## 4) 안전 규칙
- 초기화되지 않은 포인터 사용 금지
- `nullptr` 검사 후 역참조
- 해제 후 포인터를 `nullptr`로 설정

## 5) 연습 문제
1. 포인터를 통해 두 정수 값을 swap하세요.
2. 포인터 연산으로 배열 합계를 구하세요.
