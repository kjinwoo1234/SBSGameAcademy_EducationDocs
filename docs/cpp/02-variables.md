# Chapter 02 변수와 자료형

## 학습 목표
- C++ 기본 자료형과 변수 선언 방식을 익힌다.
- 상수(`const`)와 자동 형 추론(`auto`)의 차이를 이해한다.

## 세부 주제
### 02-1 변수와 자료형
- `int`, `double`, `char`, `bool`

### 02-2 상수와 초기화
- `const`, 중괄호 초기화

### 02-3 `auto`와 타입 추론
- 컴파일러의 타입 추론

## 실습 체크리스트
- 같은 값을 서로 다른 자료형으로 저장해 차이를 확인한다.

## 본문

### 02-1 변수와 자료형
- 변수는 값을 저장하는 이름 있는 메모리 공간입니다.
- 자료형은 저장할 값의 종류와 크기를 결정합니다.

기본 자료형 정리:
| 자료형 | 저장 값 예시 | 비고 |
|---|---|---|
| `int` | `10`, `-3` | 정수 |
| `double` | `3.14` | 실수 |
| `char` | `'A'` | 문자 1개 |
| `bool` | `true`, `false` | 논리값 |

### 02-2 상수와 초기화
- `const` 변수는 초기화 후 값을 바꿀 수 없습니다.
- `int n{10};` 같은 중괄호 초기화는 잘못된 축소 변환을 줄이는 데 도움이 됩니다.

### 02-3 `auto`와 타입 추론
- `auto`는 우변 초기값을 보고 타입을 자동으로 결정합니다.
- 가독성을 높일 수 있지만, 타입이 헷갈리는 코드에서는 남용을 피하는 것이 좋습니다.

용어 설명:
| 용어 | 의미 |
|---|---|
| 타입 추론(type inference) | 초기값을 바탕으로 타입을 자동 결정하는 기능 |
| 축소 변환(narrowing) | 큰 범위 타입을 작은 범위 타입으로 줄이며 값 손실이 나는 변환 |

예시 코드:
```cpp
#include <iostream>
using namespace std;

int main() {
    int level{3};
    double ratio{0.75};
    const int maxLevel{10};
    auto score = 95; // int로 추론

    cout << "level: " << level << endl;
    cout << "ratio: " << ratio << endl;
    cout << "maxLevel: " << maxLevel << endl;
    cout << "score: " << score << endl;
    return 0;
}
```

예상 출력:
```text
level: 3
ratio: 0.75
maxLevel: 10
score: 95
```

코드 해석:
- `const int maxLevel{10};`은 재할당할 수 없는 상수입니다.
- `auto score = 95;`는 초기값 `95`를 보고 `int`로 타입이 결정됩니다.
- 같은 출력문 패턴으로 서로 다른 자료형 값을 비교해 확인합니다.
