# C++ 배열

배열은 동일 타입 데이터를 고정 크기로 저장합니다.  
인덱스 접근이 빠르지만 크기를 바꾸기 어렵다는 특징이 있습니다.

## 1) 기본 사용
```cpp
int nums[5] = {10, 20, 30, 40, 50};
std::cout << nums[0] << '\n';
```

## 2) 배열 순회
```cpp
int sum = 0;
for (int i = 0; i < 5; i++) {
    sum += nums[i];
}
std::cout << sum << '\n';
```

## 3) 범위 기반 for (권장)
```cpp
for (int n : nums) {
    std::cout << n << ' ';
}
```

## 4) 실무 팁
가변 크기 컨테이너가 필요하면 `std::vector` 사용을 권장합니다.

## 5) 연습 문제
1. 배열에서 최댓값/최솟값을 구하세요.
2. 배열 평균을 소수점으로 출력하세요.
