# Chapter 05 배열과 벡터

## 학습 목표
- 고정 크기 배열과 `vector`의 차이를 이해한다.
- 반복문으로 배열/벡터를 순회할 수 있다.

## 세부 주제
### 05-1 고정 배열
- 선언, 초기화, 인덱스 접근

### 05-2 `vector` 기초
- 동적 크기 관리

### 05-3 범위 기반 for문
- 간결한 순회 문법

## 실습 체크리스트
- 학생 점수 5개를 저장하고 평균을 구한다.

## 본문

### 05-1 고정 배열
- 배열은 같은 타입 데이터를 연속된 메모리에 저장합니다.
- 인덱스는 `0`부터 시작하고, 유효 범위는 `0`부터 `크기-1`까지입니다.

### 05-2 `vector` 기초
- `vector`는 실행 중 크기를 늘리거나 줄일 수 있는 컨테이너입니다.
- `push_back`으로 원소를 뒤에 추가할 수 있습니다.

### 05-3 범위 기반 for문
- `for (int x : v)` 형태로 원소를 간단히 순회할 수 있습니다.
- 원본 수정이 필요하면 참조(`int& x`)를 사용합니다.

배열과 벡터 비교:
| 항목 | 배열 | `vector` |
|---|---|---|
| 크기 변경 | 불가(고정) | 가능(가변) |
| 문법 난이도 | 단순 | 기능이 많아 유연 |
| 권장 사용 | 크기가 고정된 데이터 | 크기가 바뀌는 데이터 |

용어 설명:
| 용어 | 의미 |
|---|---|
| 컨테이너(container) | 여러 데이터를 저장/관리하는 자료구조 객체 |
| 범위 기반 for문 | 컬렉션 전체를 순회하는 간결한 반복문 |

예시 코드:
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int arr[3] = {10, 20, 30};
    vector<int> v = {1, 2, 3};
    v.push_back(4);

    cout << "arr: ";
    for (int i = 0; i < 3; i++) cout << arr[i] << ' ';
    cout << "\nvector: ";
    for (int x : v) cout << x << ' ';
    cout << endl;
    return 0;
}
```

예상 출력:
```text
arr: 10 20 30
vector: 1 2 3 4
```

코드 해석:
- `arr`는 고정 크기 배열이라 선언 시 크기가 정해집니다.
- `vector<int> v`는 `push_back(4)`로 실행 중 원소를 추가할 수 있습니다.
- 두 컨테이너를 각각 일반 `for`와 범위 기반 `for`로 순회해 차이를 보여줍니다.

- 인덱스 범위 초과 접근 금지
