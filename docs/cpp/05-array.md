# Chapter 05 배열과 벡터

## 학습 목표
- 고정 크기 배열을 선언·초기화하고 인덱스로 접근한다.
- `vector`로 크기가 변하는 목록을 다루는 기초를 익힌다.
- 일반 `for`와 범위 기반 `for`로 순회할 수 있다.

## 본문

### 05-1 고정 배열 선언

변수는 값 하나를 담는 상자에 가깝습니다. 같은 종류의 값이 여러 개면 **배열**로 모읍니다. **배열**은 같은 자료형의 값을 **연속된 칸**에 모아 두고, 이름 하나로 그 칸들을 다루는 자료 구조입니다. 여기서 배우는 고정 배열은 선언할 때 칸 개수가 정해지고, 실행 중에 늘리거나 줄일 수 없습니다.

먼저 칸 개수만 정해 만드는 방법입니다. 아래 한 줄의 의미를 하나씩 짚어 보겠습니다.

```cpp
int scores[5];
```

- `int` — 각 칸에 들어갈 자료형입니다.
- `scores` — 배열의 이름입니다.
- `[5]` — 칸이 **5개**라는 뜻입니다.

`main` 안의 지역 배열은 값을 넣기 전까지 내용이 정해져 있지 않을 수 있습니다. 칸을 만든 뒤 `scores[0] = 80;`처럼 값을 넣습니다.

아래를 실행해 보세요.

```cpp
#include <iostream>
using namespace std;

int main() {
    int scores[5];
    scores[0] = 80;
    scores[1] = 95;
    cout << scores[0] << endl;
    return 0;
}
```

**예상 출력**

```text
80
```

**코드 해석**
- `int scores[5];`로 칸 5개를 만든 뒤 `scores[0]`에 `80`을 넣습니다.
- `scores[0]`을 출력하면 `80`이 나옵니다.

이어서 처음부터 값을 넣어 만드는 방법입니다. **초기화 목록**을 씁니다. 아래 한 줄의 의미를 하나씩 짚어 보겠습니다.

```cpp
int scores[] = { 60, 85, 70, 90 };
```

- `int scores[]` — 정수 배열 `scores`를 선언합니다. `[]` 안 크기를 비우면 오른쪽 값 개수로 크기가 정해집니다.
- `{ 60, 85, 70, 90 }` — 처음부터 넣을 값을 중괄호 안에 나열합니다.

값이 네 개이므로 칸도 네 개입니다.

아래를 실행해 보세요.

```cpp
#include <iostream>
using namespace std;

int main() {
    int scores[] = { 60, 85, 70, 90 };
    cout << scores[0] << endl;
    return 0;
}
```

**예상 출력**

```text
60
```

**코드 해석**
- 첫 칸 `scores[0]`은 `60`입니다.

### 05-2 칸 개수 확인

고정 배열은 칸 개수를 이름만으로 바로 알려 주지 않습니다. 대신 **`sizeof`**로 계산합니다. `sizeof(scores) / sizeof(scores[0])`가 칸 개수입니다.

아래를 실행해 보세요.

```cpp
#include <iostream>
using namespace std;

int main() {
    int a[5];
    int b[] = { 60, 85, 70, 90 };

    cout << sizeof(a) / sizeof(a[0]) << endl;
    cout << sizeof(b) / sizeof(b[0]) << endl;
    return 0;
}
```

**예상 출력**

```text
5
4
```

**코드 해석**
- `a`는 `[5]`이므로 칸 개수는 `5`입니다.
- `b`는 값이 네 개이므로 칸 개수는 `4`입니다.

### 05-3 인덱스와 순회

**인덱스**는 각 칸의 위치 번호이며 **0부터** 시작합니다. 세 칸이면 유효 번호는 `0`, `1`, `2`입니다. 범위를 벗어나면 잘못된 동작이 날 수 있으므로 `0`부터 `크기 - 1`까지만 접근합니다.

아래는 인덱스를 쓰는 `for`로 배열을 순회하는 예입니다.

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = { 10, 20, 30 };
    int n = (int)(sizeof(arr) / sizeof(arr[0]));

    for (int i = 0; i < n; i++) {
        cout << arr[i] << ' ';
    }
    cout << endl;
    return 0;
}
```

**예상 출력**

```text
10 20 30
```

**코드 해석**
- `i`는 `0`부터 `n` 직전까지입니다.
- `arr[i]`로 각 칸을 출력합니다.

### 05-4 `vector` 기초

크기가 실행 중에 바뀌어야 하면 고정 배열 대신 **`vector`**를 씁니다. `vector`는 원소를 담는 **컨테이너**이고, `push_back`으로 뒤에 값을 추가할 수 있습니다. 칸 개수는 **`.size()`**로 확인합니다.

아래 한 줄의 의미를 짚어 보겠습니다.

```cpp
vector<int> v = { 1, 2, 3 };
```

- `vector<int>` — `int`를 담는 벡터입니다.
- `v` — 이름입니다.
- `{ 1, 2, 3 }` — 처음부터 넣을 값입니다.

아래를 실행해 보세요.

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v = { 1, 2, 3 };
    v.push_back(4);
    cout << v.size() << endl;
    cout << v[0] << endl;
    return 0;
}
```

**예상 출력**

```text
4
1
```

**코드 해석**
- 처음 원소 세 개에 `push_back(4)`로 하나 더해 크기가 `4`가 됩니다.
- `v[0]`은 첫 값 `1`입니다.

| 항목 | 배열 | `vector` |
|---|---|---|
| 크기 변경 | 불가(고정) | 가능(가변) |
| 칸 개수 확인 | `sizeof`로 계산 | `.size()` |
| 권장 사용 | 크기가 고정된 데이터 | 크기가 바뀌는 데이터 |

### 05-5 범위 기반 for문

값만 순서대로 읽을 때는 **범위 기반 for**가 짧습니다. 형태는 `for (int x : v)`입니다. `int x`는 꺼낸 값을 담을 변수이고, `: v`는 어느 컨테이너에서 꺼낼지입니다. 원본 칸을 바꾸려면 `int& x`처럼 참조를 씁니다.

아래를 실행해 보세요.

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v = { 1, 2, 3, 4 };

    for (int x : v) {
        cout << x << ' ';
    }
    cout << endl;
    return 0;
}
```

**예상 출력**

```text
1 2 3 4
```

**코드 해석**
- `v`의 값을 순서대로 `x`에 담아 출력합니다.
- 인덱스가 필요 없을 때 쓰기 좋습니다.

| 용어 | 의미 |
|---|---|
| 컨테이너 | 여러 데이터를 저장·관리하는 자료구조 객체 |
| 범위 기반 for문 | 컬렉션 전체를 순회하는 간결한 반복문 |
