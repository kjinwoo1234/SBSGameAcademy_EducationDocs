# SVN 자습 자료

## 권장 대상
- 중앙 저장소 기반 레거시 프로젝트를 유지/운영하는 학습자
- 사내 협업 환경에서 SVN 워크플로우를 익혀야 하는 학습자

## 선수지식
- 파일/폴더 관리와 CLI 기본 사용
- 버전 관리의 기본 개념(커밋, 충돌, 병합)

## 완주 후 역량
- SVN 저장소 체크아웃부터 커밋/업데이트까지 기본 흐름을 수행할 수 있다.
- 브랜치/태그 구조를 이해하고 릴리스 관리에 적용할 수 있다.
- 충돌 해결 후 안정적으로 작업본을 정리할 수 있다.

## 시작 전 체크
- SVN 클라이언트 설치 후 `svn --version` 확인을 완료했는가?
- 저장소 URL과 계정 권한을 확인했는가?

## 완주 체크
- `checkout -> update -> commit` 흐름을 반복 수행할 수 있는가?
- `trunk/branches/tags` 구조의 목적을 설명할 수 있는가?
- 충돌 해결 후 상태를 정상으로 되돌릴 수 있는가?
- 릴리스 태그 생성과 검증 기록까지 포함한 운영 절차를 재현할 수 있는가?
